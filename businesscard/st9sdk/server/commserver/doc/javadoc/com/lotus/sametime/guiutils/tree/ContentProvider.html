Interface ContentProvider

  • All Known Implementing Classes:

    due to the availability of additional non-AWT frameworks. This AWT component is no longer supported.

    public interface ContentProvider
    An interface by which content is provided for a column display in a multi-column list that displays attributes. The content provided could be an image, string or null for an empty display.
