com.lotus.sametime.cachedresolvemanager.resolve

Class SametimeResolver

  • java.lang.Object
    • com.lotus.sametime.cachedresolvemanager.resolve.SametimeResolver
  • All Implemented Interfaces:
    ResolveListener, java.util.EventListener

    public class SametimeResolver
extends java.lang.Object
implements ResolveListener
    Using Resolver object sends resolve requests to the ST server and listen to the resolve responses. If the ST cluster is configured, this class instance manages a number of Resolver objects (per ST server in the cluster) and chooses a Resolver instance for the current resolve request as per the round-robin algorithm. Receives the resolve notification events from the ST server and forwards them to the CachedResolveImpl class instance for further processing.
