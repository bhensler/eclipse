createProviderPropertiesChangedEvent

public abstract ProviderPropertiesChangedEvent createProviderPropertiesChangedEvent(java.lang.String providerId,
                                                                                    java.lang.String communityId,
                                                                                    java.util.Properties properties,
                                                                                    java.util.Set<java.lang.String> capabilities)
Creates a ProviderPropertiesChangedEvent instance for the specified service provider, used to indicate a change to a provider's TCSPI configuration properties or capabilities. If any properties have changed, the properties parameter should only include the updated properties. If any capabilities have changed, the capabilities parameter should include the complete set of supported capabilities, not just the additional capabilities. Sets the event type to EVENT_PROVIDER_PROPERTY_CHANGE.

Parameters:
providerId - service provider ID, cannot be null
communityId - service provider community ID, cannot be null
properties - updated configuration properties, null if none changed
capabilities - complete set supported capabilities, null if none changed
Returns:
New ProviderPropertiesChangedEvent instance.
Since:
Sametime 8.5.2
