Class SelectiveStatusList

  • java.lang.Object
    • com.lotus.sametime.selectivestatus.SelectiveStatusList

  • public class SelectiveStatusList
extends java.lang.Object
    A list of users and groups who see the list owner's default (trigger) status as a different status specified by userDisplayStatus.

      • dump

        public void dump(NdrOutputStream os)
          throws java.io.IOException
        Dump the list to an NdrOutputStream based on the protocol for the SelectiveStatus.

        List flag.
        Right bit isPersistant (0x0001)
        second bit hasNames (0x0002)
        third bit overrides privacy (0x0004)

        Parameters:
        os - - an Ndr outputstream containining the dumped list.
        Throws:
        java.io.IOException
        Since:
        Sametime 8.5
