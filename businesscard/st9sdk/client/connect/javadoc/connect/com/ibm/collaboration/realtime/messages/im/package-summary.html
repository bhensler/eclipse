Package com.ibm.collaboration.realtime.messages.im

Class Summary
ImChatWindowCloseWarningMessage ImChatWindowCloseWarningMessage is generated by plug-ins to enable or disable a warning dialog when the user closes the chat window.
ImChatWindowFlashWindowMessage ImChatWindowFlashWindowMessage is generated by plug-ins to request the UI code to flash the specified chat window.
ImChatWindowForceFocusMessage ImChatWindowForceFocusMessage is generated by plug-ins to request the UI code to set focus on the specified chat window.
ImChatWindowToolbarRedrawMessage ImChatWindowToolbarRedrawMessage is generated by plug-ins to request the UI code to redraw the specified chat window toolbar.
ImConnectedMessage ImConnectedMessage is generated when the current user logs in to a community.
ImConnectionStatusLocationChangeMessage ImConnectionStatusLocationChangeMessage when a new location is detected for the current user.
ImDisconnectedMessage ImDisconnectedMessage is generated when the current user logs out of a community.
ImMessage Abstract base class for message event classes related to instant messaging.
ImTextConnectionClosedMessage ImTextConnectionClosedMessage is generated when a 2-way chat ends.
ImTextConnectionOpenMessage ImTextConnectionOpenMessage is generated by a plug-in to request the UI code to open a 2-way chat window.
ImTextConnectionOpenSuccessMessage ImTextConnectionOpenSuccessMessage is generated when a 2-way chat window is opened successfully.
ImTextConversionManyToManyMessage ImTextConversionManyToManyMessage is generated when a 2-way chat window is converted to an n-way chat window.
ImTextReceivedMessage ImTextReceivedMessage is generated when text is received in a 2-way chat.
ImTextSendFailedMessage ImTextSendFailedMessage is generated when an attempt to send text in a 2-way chat fails.
ImTextSendMessage ImTextSendMessage is generated to send text in a 2-way chat.
ImTextTypingMessage ImTextTypingMessage is generated when a chat partner starts or stops typing in a chat.
ImTextTypingRequestMessage ImTextTypingRequestMessage is generated when the current user starts or stops typing in a chat.
ImWindowClosedMessage ImWindowClosedMessage is generated when a 2-way or n-way chat window closes.
ImWindowOpenedMessage ImWindowClosedOpened is generated when a 2-way or n-way chat window opens.
ManyToManyCreateMessage ManyToManyCreateMessage is generated by a plug-in to create a new n-way chat window.
ManyToManyCreateSuccessMessage ManyToManyCreateSuccessMessage is generated when an n-way chat window is successfully created.
ManyToManyInviteAcceptMessage ManyToManyInviteAcceptMessage is generated when an invitation to an n-way chat is accepted.
ManyToManyInviteDeclineMessage ManyToManyInviteDeclineMessage is generated when an invitation to an n-way chat is declined.
ManyToManyInviteMessage ManyToManyInviteMessage is generated when an invitation to an n-way chat is received.
ManyToManyInviteOthersMessage ManyToManyInviteOthersMessage is generated when the current user invites other users to an n-way chat.
ManyToManyLeaveMessage ManyToManyLeaveMessage is generated when the current user leaves an n-way chat, for example, by closing the chat window.
ManyToManyTextConnectionClosedMessage ManyToManyTextConnectionClosedMessage is generated when an n-way chat ends.
ManyToManyTextReceivedMessage ManyToManyTextReceivedMessage is generated when text is received in an n-way chat.
ManyToManyTextSendMessage ManyToManyTextSendMessage is generated when text is sent in an n-way chat.
ManyToManyUserEnterExitMessage ManyToManyUserEnterExitMessage is generated when a user enters or leaves an n-way chat.
PartnerStatusUpdateMessage PartnerStatusUpdateMessage is generated when a chat partner's status changes.
