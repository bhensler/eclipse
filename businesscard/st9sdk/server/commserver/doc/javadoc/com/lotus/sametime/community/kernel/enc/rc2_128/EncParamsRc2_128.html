Class EncParamsRc2_128

  • java.lang.Object
    • com.lotus.sametime.community.kernel.enc.EncParams
      • com.lotus.sametime.community.kernel.enc.rc2_128.EncParamsRc2_128

  • public class EncParamsRc2_128
extends EncParams
    Parameters for the Rc2-128 encryption method. This class is responsible for the Diffie-Hellman key exchange. The various states for this object. An EncParamsRc2_128 object can be used to store the local private and public keys, or it can be used to store the other side's public key. The agreed key is kept as part of the local object.
