Class STBaseImpl

  • java.lang.Object
    • com.lotus.sametime.community.kernel.VpKernel
      • com.lotus.sametime.community.STBaseImpl
  • All Implemented Interfaces:
    CnlMsgListener, MasterCnlListener, STEventListener, TimerListener

    public class STBaseImpl
extends VpKernel
implements STEventListener, TimerListener
    STBase component implementation. This class inherits from VpKernel, which actually supplies the functionality, and adds whatever is needed to make STBase a component. Because it has to inherit from VpKernel, this class does not extend STCompImpl. Instead, we implement STEventListener, and supply all the functionality ourselves.

      • loginWithoutCredentials

        public boolean loginWithoutCredentials(java.lang.String host,
                              java.lang.String loginName,
                              java.lang.String community,
                              ConnectionHandler conHandler,
                              java.net.InetAddress ip,
                              java.lang.String location)
        Requests to perform login without credentials to the community.
        Parameters:
        host - DNS name of server/mux to connect to.
        loginName - user login name.
        community - The community to connect to (null or an empty string can be used if there's only one community on the server).
        conHandler - The connection handler to use for this login. must not be null.
        ip - The IP to report to the server. This can be used by mux applications to do a login for a client. If null, (as it usually should be) the local IP will be used.
        location - The location to report to the server. The location when available is used the mux/server to determine the machine running the application. The mux/server decides according to the ip & location when to disconnect clients running on different machines/locations. If null, the local host name will be used.
