Class CommunityEventsComp

  • java.lang.Object
    • com.lotus.sametime.core.comparch.STCompPart

  • All Implemented Interfaces:
    ChannelListener, LoginListener, CommunityEventsService, STCompApi, java.util.EventListener

    public class CommunityEventsComp
extends STComp
implements CommunityEventsService, ChannelListener, LoginListener
    A component implementing the comm events protocol. We open a channel for every server in the community (see the serviceUp impl), and pass all subscribe requests to all of the servers. The only exception is the serviceUp/Down subscription which are global.
