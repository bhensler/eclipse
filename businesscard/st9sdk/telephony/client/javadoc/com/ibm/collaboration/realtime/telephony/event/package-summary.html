Package com.ibm.collaboration.realtime.telephony.event

Interface Summary
CallEvent CallEvent represents a call-related telephony event, processed by CallListener.processCallEvent(CallEvent).
CallPropertiesEvent CallPropertiesEvent is a call event used to provide properties associated with a call.
CallStartedEvent CallStartedEvent is a call event used to signal that a call has started.
CapabilityEvent CapabilityEvent is used to indicate that a call capability is supported or no longer supported.
IncomingCallEvent IncomingCallEvent is a call event used to signal an incoming call.
ParticipantAddedEvent ParticipantAddedEvent is a call event used to provide information about a new participant that was added to the call.
PropertyChangeEvent PropertyChangeEvent is a call event used to signal a change to the properties of the associated call or one of the call participants.
ProviderPropertiesChangedEvent ProviderPropertiesChangedEvent is used by a service provider to indicate a change to the provider's configuration properties and/or capabilities.
ProviderPropertiesEvent ProviderPropertiesEvent is used to publish service provider configuration properties and capabilities, when the client first receives the configuration from the provider.
ServiceErrorEvent ServiceErrorEvent is a call event used to signal an error from the telephony service provider.
ServiceEvent ServiceEvent represents a telephony event related to the telephony service, rather than a particular call.
TelephonyEvent TelephonyEvent is the base interface for all telephony API events.
 

Class Summary
CallEventFactory CallEventFactory is used to create event objects.
