CAPABILITY_PRODUCT_VERSION

static final java.lang.String CAPABILITY_PRODUCT_VERSION
Determines which product build the person is running, for users running Sametime 8.0 or later. The format of the value is "release,build", where the release and build are the strings returned by the getRelease() and getBuild() methods of ProductVersion. If the build is unknown, only the release is included (without the separator). If the release is unknown, the value will be CAPABILITY_PRODUCT_VERSION_UNKNOWN, in which case it's safe to assume the person is running Sametime 8.0 or a later release.

Since:
Sametime 8.0

getCapabilities

java.util.Map getCapabilities(java.lang.String loginId)
Returns a Map of capabilities associated with the specified person's instant messaging client login instance. Each capability is represented by a key/value pair, where the key is one of the CAPABILITY_xxx constants defined in this interface, and the value is a string, typically "true" (capability supported) or "false" (capability not supported). Any capability that has a value other than "false" is assumed to be supported. Any capability that is not present in the Map is assumed to be unsupported. To simply determine if a specific capability is supported, use isCapabilitySupported(String, String) instead; see that method for additional information about the login instance.

Parameters:
loginId - login instance ID
Returns:
Map of capabilities for the specified login instance ID, null if not available.

isCapabilitySupported

boolean isCapabilitySupported(java.lang.String loginId,
                              java.lang.String capability)
Determines if the specified capability is supported by the specified person's instant messaging client login instance. See the CAPABILITY_xxx constants defined in this interface for valid capability key names. Note that a person can be logged in with different IM clients at the same time, and each login may have a different set of capabilities. You can obtain a chat partner's login instance ID using the ChatWindowPartner.getPersonLoginId() method; to get a ChatWindowPartner object, use one of the methods in ChatWindowHandler.

Parameters:
loginId - login instance ID
capability - capability key name, one of the CAPABILITY_xxx constants defined in this interface
Returns:
true if the capability is supported, false if not supported

getGatewayCommunity

java.lang.String getGatewayCommunity()
Returns the RTC gateway community name to which this person belongs. The gateway community is only relevant if this person is an external user, and may be null if the person belongs to an external sametime community. It may be non-null if the person belongs o public IM community such as AOL, google, or yahoo.
