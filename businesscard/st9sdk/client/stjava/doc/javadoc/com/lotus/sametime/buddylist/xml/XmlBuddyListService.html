Interface XmlBuddyListService

  • All Superinterfaces:
    STCompApi
    All Known Implementing Classes:
    XmlBuddyListMgr

    public interface XmlBuddyListService
extends STCompApi
    XmlBuddyListService is used to manage an instant messaging contact ("buddy") list.

    Note: This is the new XML format BuddyList that the Sametime 7.5+ clients use.

    Note: The XmlBuddyList provides support for multiple communities. Be careful to only subscribe (or watch) users that belong to the current community or communities that you are logged into.
    Since:
    Sametime 8.0.2

      • setSynchronizeBuddyLists

        void setSynchronizeBuddyLists(boolean synchronize)
        Specify whether the XmlBuddyListService should synchronize all buddylist activities between the new XML format and the old classic format. If synchronization is enabled, you can expect the following behavior: - XmlBuddyLists when retrieved from storage using the @see getBuddyList() method will be compared to the classic version and synchronized if necessary. - Changes to XML will be stored to the classic format in storage when @see saveBuddyList(XmlBuddyList) is called.
        Parameters:
        synchronize - true to enable synchronization of buddylists, false otherwise. The default is true.
        Since:
        Sametime 8.5
