com.lotus.sametime.core.types

Class STPrivacyList

  • java.lang.Object
    • com.lotus.sametime.core.types.STPrivacyList
  • All Implemented Interfaces:
    java.lang.Cloneable

    public class STPrivacyList
extends java.lang.Object
implements java.lang.Cloneable
    This class represents a Sametime privacy list which is either excluding or including. An excluding privacy list is a list of users who cannot see me online. An including privacy list is a list of users who can see me online.
