Class ServiceHub

java.lang.Object
  com.ibm.collaboration.realtime.servicehub.ServiceHub

public class ServiceHub
extends java.lang.Object

ServiceHub provides static methods to access instances of Sametime services. Services can be registered in one of two ways:

1. Extending the com.ibm.collaboration.realtime.core.services extension point. 2. Using the registerService(String, Object) method in this class.

Returns the implementation for the given service name. If a single service implementation exists, it is returned. If multiple service implementations exist, the implementation marked default is returned. If there is no default service, or if multiple services are marked default, a ServiceException is thrown.
