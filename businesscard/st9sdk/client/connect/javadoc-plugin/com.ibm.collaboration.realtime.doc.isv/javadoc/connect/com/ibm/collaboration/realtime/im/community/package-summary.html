Package com.ibm.collaboration.realtime.im.community

Interface Summary
AuthTypeProvider This interface is part of the com.ibm.collaboration.realtime.community.authTypes extension point and provides an authentication provider various hooks to control authentication behavior.
Community Community represents a instant messaging and presence community.
CommunityListener Listener for community-related events.
CommunityService CommunityService is used to access and manage communities.
 

Class Summary
CommunityEvent CommunityEvent is the base class for events used by the community APIs.
CommunityLifecycleEvent CommunityLifecycleEvent is generated when a community lifecycle event occurs, for example, when a community is created, deleted, or updated.
CommunityLoginEvent CommunityLoginEvent is generated when a community login or logout event occurs for the current user.
CommunityServiceEvent CommunityServiceEvent is generated when a service-related community event occurs, for example, when a service is loaded for a particular community.
CommunityStatusEvent CommunityStatusEvent is generated when the current user's status changes.
 

Exception Summary
CommunityException CommunityException is the base class for exceptions used by the community APIs.
CommunityLoginException CommunityLoginException is thrown by the community APIs if a login attempt fails.
