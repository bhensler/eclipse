Class AwarenessViewController

  • java.lang.Object
    • com.lotus.sametime.awarenessui.AwarenessViewController
  • Direct Known Subclasses:
    ChatController, STTreeController
    Deprecated. 
    due to the availability of additional non-AWT GUI frameworks. This component is no longer supported.

    public abstract class AwarenessViewController
extends java.lang.Object
    Base class for all awareness view controllers. The controller's task is to supply custom behavior for user actions. The controller controls the popup menus displayed to the user according to the list context and performs the operations as a result of the menu item being selected. Each view has a default controller associated with it but the controller may be replaced to modify the list's behavior.
