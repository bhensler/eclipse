com.ibm.collaboration.realtime.telephony.device
Interface DeviceEvent

All Superinterfaces:
java.io.Serializable, TelephonyEvent
public interface DeviceEvent
extends TelephonyEvent

DeviceEvent represents a device-related telephony event, processed by DeviceListener.processDeviceEvent(DeviceEvent). When necessary, use getDeviceName() to identify which device the event is associated with. Note that the EVENT_xxx constants defined in this interface specify all the event types that are part of the API, but service providers can define additional event types if desired.

Since:
Sametime 7.5.1
