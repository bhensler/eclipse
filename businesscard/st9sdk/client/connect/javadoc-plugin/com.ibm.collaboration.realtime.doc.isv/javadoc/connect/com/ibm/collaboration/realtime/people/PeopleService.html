getPerson

Person getPerson(java.lang.String contactId,
                 java.lang.String communityId,
                 java.lang.String gatewayCommunityId,
                 boolean isExternal)
Returns the specified person, retrieving from the person cache. The person is retrieved from the cache if present there, otherwise the peron is created and added to the cache. The specified person can be an external user connected via a gateway. If the person is external, a gateway community ID may be specified.

Parameters:
contactId - contact ID of person to retrieve
communityId - ID of community to retrieve person from
gatewayCommunityId - ID of the gateway community to which the external user belongs, or null.
isExternal - true if the person is an external (gatewayed) user, false otherwise
Returns:
Person for the specified person.
