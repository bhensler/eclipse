Interface MediaQualityServiceProvider

All Known Implementing Classes:
AbstractMediaQualityServiceProvider
public interface MediaQualityServiceProvider

MediaQualityServiceProvider is implemented by the plug-in that provides media quality evaluation services for the Sametime Connect client. Note: The user/administrator must explicitly enable media quality evaluation by setting the com.ibm.collaboration.realtime.telephony.ui/enableMediaQualityServiceProvider preference to true via plugin_customization.ini or a contributed preferences page. This "enable" preference is necessary to avoid the overhead of loading the plug-in unless it's actually needed (by default, it is not loaded).

Since:
Sametime 8.5.2
