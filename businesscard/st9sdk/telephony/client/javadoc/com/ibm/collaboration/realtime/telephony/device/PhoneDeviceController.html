setCallForwardingEnabled

void setCallForwardingEnabled(boolean flag)
                              throws TelephonyServiceException
Deprecated. 
Enables or disables call forwarding, if supported by the service provider. Before enabling call forwarding, call forwarding options must be specified either on the phone device or by using setCallForwardingOptions(CallForwardingOptions). Call forwarding can also be enabled or disabled when using the setCallForwardingOptions method, so it is not necessary to use both methods together.

Parameters:
flag - true to enable call forwarding, false to disable
Throws:
UnsupportedCapabilityException - if this operation is not supported
InvalidOperationException - if you enable call forwarding and no options have been specified
TelephonyServiceException - if the operation fails for some other reason
