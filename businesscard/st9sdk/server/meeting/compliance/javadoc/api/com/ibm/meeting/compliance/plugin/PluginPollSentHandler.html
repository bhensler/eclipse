pollSent

void pollSent(PollSentEvent event)
Called when a poll is entered in the meeting room. Recipient plugins may disallow the poll by setting event.status to EVENT_BLOCKED_TYPE, optionally also writing a message to event.reason to explain the block. Furthermore, the poll may be modified, and the event.status may be set to EVENT_MODIFIED_TYPE

Parameters:
event - contains action details such as the user, original question, and status fields for reporting the plugin's response

onPollSent

void onPollSent(PollSentEventStatus status)
Called after an attempted poll action has been approved or denied (indicated by EVENT_OK, EVENT_MODIFIED, or EVENT_BLOCKED in the status field). Primarily used by plugins for activity logging.

Parameters:
status - contains action details including user and question, whether the action was blocked.
