Class PublishingUserImpl

  • java.lang.Object
    • com.lotus.sametime.telephonymanager.publishing.PublishingUserImpl
  • All Implemented Interfaces:
    LoginListener, PublishingUser, java.util.EventListener

    public class PublishingUserImpl
extends java.lang.Object
implements PublishingUser, LoginListener
    A PublishingUser implementation for ST80 server versions (and above?) Life Cycle : When this object is created we request a server to login to (from the cluster), and do the login. When we receive the loggedIn event, we save the server that we were logged in to and send it a publish request for the current telephony status of this user. If we receive a loggedOut event, it is our responsibility to try to login again to another server.
