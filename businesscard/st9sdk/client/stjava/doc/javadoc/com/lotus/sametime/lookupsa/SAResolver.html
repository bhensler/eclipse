com.lotus.sametime.lookupsa

Class SAResolver

  • java.lang.Object
    • com.lotus.sametime.lookupsa.SAResolver

  • public class SAResolver
extends java.lang.Object
    A name resolver. Use this class to resolve user or group names to STuser or STgroup objects as registered in the directory of the Sametime server. Unlike the client-side resolver, this resolver can look in different organization address books.

        organization - The organization address book to look in.
