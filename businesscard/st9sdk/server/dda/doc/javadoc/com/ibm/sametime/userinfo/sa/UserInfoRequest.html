Class UserInfoRequest

  • java.lang.Object
    • com.ibm.sametime.userinfo.sa.UserInfoRequest
  • Direct Known Subclasses:
    UserInfoRequestHTTP, UserInfoRequestVP

    public abstract class UserInfoRequest
extends java.lang.Object
    The request object is being used for storing received parameters from a userinfo request. This object is then added to the message queue. it is being removed from there by a worker thread which activates the Process of data retrieval and then sends the info received to the requester.
