com.lotus.sametime.userattributesa

Interface UserAttributeSAService

  • All Superinterfaces:
    STCompApi
    All Known Implementing Classes:
    UserAttributeSAComp

    public interface UserAttributeSAService
extends STCompApi
    The UserAttributeSAService enable server side components to set and remove attributes associated with specific users. These attributes are made visible to other users and server application by the awareness service (i.e. buddy list). For the viewer watching these attributes, they look the same as the attributes that the user/client sets by itself. This service enables the creation of a 3rd-party server-side component, which can contribute additional information to the current state of the user, e.g., indicating that the user is currently on the phone by receiving the information from its PBX. Only server-side components have permission to utilize this service; client-side logins cannot change attributes for other users in the system.
