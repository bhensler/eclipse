Class StIcon

  • java.lang.Object
    • com.lotus.sametime.guiutils.sttable.StIcon
  • Deprecated. 
    due to the availability of additional non-AWT frameworks. This AWT component is no longer supported.

    public class StIcon
extends java.lang.Object
    A class that encapsulates Image object. This class provides a static method for image fetching. For the classes that use Image object, this class provides a public getter getImage().
    The painting of the Icon should be done through the public 
     paintIcon(Graphics g, <font color='blue'>int</font> x, 
    <font color='blue'>int</font> y, Component c)
    call.
