Interface OrgNodeProvider

All Superinterfaces:
Validatable
public interface OrgNodeProvider
extends Validatable

The OrgNodeProvider is used by the OrgTreeProvider to create OrgNode objects from directory search results and can be thought of as a factory for OrgNode objects.

Since:
