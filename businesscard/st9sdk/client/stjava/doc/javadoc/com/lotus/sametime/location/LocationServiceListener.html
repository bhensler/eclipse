      • locationChanged

        void locationChanged(LocationEvent event)
        A Location Service notification that a watchlist user's Location has changed. Use event.getLocation() to obtain the new Location for the watched user, event.getLocation().getSTUser().
        Parameters:
        event - - LocationEvent.

      • locationRemoved

        void locationRemoved(LocationEvent event)
        A Location Service notification that a watchlist user's Location has been removed. Use event.getLocation().getSTUser() to determine the watchlist user. No other event.getLocation() data is valid.
        Parameters:
        event - - LocationEvent
