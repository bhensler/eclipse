com.ibm.collaboration.realtime.telephony
Interface EndpointDescriptor

public interface EndpointDescriptor

EndpointDescriptor describes a telephony endpoint, and is used to display preferred telephone numbers or devices in the Sametime Connect user interface.

Since:
Sametime 8.0

getKey

java.lang.String getKey()
Returns the unique key for this endpoint descriptor. The key is used to identify this endpoint with other API methods that use endpoint descriptors, and is either generated automatically or specified by the service provider when the endpoint descriptor is created. The value of the key has no significance other than to identify this endpoint descriptor, and does not change once created.

setAddressUri

void setAddressUri(java.lang.String uri)
Sets the address URI of this endpoint. The address URI specifies the phone number or other endpoint used to call a user, in RFC3966 format. When making a call or adding participants to an existing call, the address URI is passed to the service provider via an Address object. See Address.setAddressUri(String) for more information.

Parameters:
uri - address URI, cannot be null
Throws:
InvalidParameterException - if an invalid URI is specified

setDescription

void setDescription(java.lang.String description)
Sets the description of this endpoint. The description, if specified, is displayed as tooltip text for the preferred device drop-down control. If no description is provided, the tooltip is based on the label and displayable value.

Parameters:
description - endpoint description or null

setDisplayValue

void setDisplayValue(java.lang.String value)
Sets the displayable value of this endpoint, typically the phone number or other string as entered by the user, but it can also be specified by the service provider. This value is used in conjunction with the label when displaying the endpoint in the user interface, and is recommended to be no longer than necessary. If no displayable value is specified, the address URI is used instead.

Parameters:
value - displayable value or null

setIsComputer

void setIsComputer(boolean flag)
Specifies whether this endpoint represents a computer device. A computer endpoint might also be a telephone device that can make and receive telephone calls, for example, a softphone. The default value is false.

Parameters:
flag - true if this endpoint is a computer device, false (the default) otherwise
Since:
Sametime Unified Telephony 8.0, Sametime 8.0.2

setIsTelephone

void setIsTelephone(boolean flag)
Specifies whether this endpoint represents a telephone device that can make and receive telephone calls. The device might be a physical phone or a computer softphone. The default value is true.

Parameters:
flag - true (the default) if this endpoint is a telephone device, false otherwise

setIsUserDefined

void setIsUserDefined(boolean flag)
Specifies whether this endpoint descriptor was created by the end user, using the Preferred Number preferences page. Endpoint descriptors are created by either the user, Sametime Connect, or the service provider. The Sametime Connect user interface doesn't allow users to edit or remove endpoints that they didn't defined. The default is true.

Parameters:
flag - true (the default) if this is a user-defined endpoint, false otherwise.

setIsVideoDevice

void setIsVideoDevice(boolean flag)
Specifies whether this endpoint represents a device with video capabilities. Both telephone and non-telephone devices can have video capabilities, so this setting is independent of the isTelephoneDevice setting. Specify true only if the service provider supports video when using this device. The default value is false.

Parameters:
flag - true if this endpoint is a video device, false (the default) otherwise

setIsVoicemailDevice

void setIsVoicemailDevice(boolean flag)
Specifies whether this endpoint represents a telephony device designated for voicemail. In general, this is a system-defined device to which calls are routed if the user doesn't answer a call or elects to direct a call to voicemail. The default value is false.

Parameters:
flag - true if this endpoint represents a voicemail device, false (the default) otherwise
Since:
Sametime Unified Telephony 8.0, Sametime 8.0.2

setLabel

void setLabel(java.lang.String label)
Sets the label associated with this endpoint, for example, Work, Home, Computer, etc. The label might be retrieved from the directory, or specified by the service provider or user.

Parameters:
label - endpoint label or null

setNoAnswerSeconds

void setNoAnswerSeconds(int seconds)
Sets the number of seconds the system should wait for an answer when calling this endpoint. The default is 10 seconds. If the user doesn't answer the endpoint, the call routing conditions determine what happens next.

Parameters:
seconds - number of seconds to wait for an answer when calling this endpoint
Since:
Sametime Unified Telephony 8.0, Sametime 8.0.2
