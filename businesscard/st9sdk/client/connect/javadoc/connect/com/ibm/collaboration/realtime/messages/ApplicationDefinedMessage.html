public class ApplicationDefinedMessage
extends Message

Custom application-defined message for third-party use. Use Message.setProperty(String, Object) to specify any properties you wish to include in the message. Use Message.setSourceId(String) to specify a unique identifier for your plug-in, for example, plug-in ID or class name, so that you can distinguish your ApplicationDefinedMessage objects from those created by other plug-ins.
