A static method, to get the server Configuration, using the currently connected client's ServiceBase. Use class constants, defined in the ConfigurationKeys interface to pull values. 
        RoomServices roomSvcs = new RoomServices(SERVER_URL);
        Configuration config = Configuration.getServerConfig(roomSvcs);
        String stringValue = config.getStringValue(Configuration.MRC_CUSTOM_LOGIN_PAGE);
        boolean booleanValue = config.getBooleanValue(Configuration.MRC_ALLOW_GUEST_ACCESS);
        int intValue = config.getIntValue(Configuration.DS_JPEG_QUALITY);
