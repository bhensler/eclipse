Interface SelectiveStatusListener


  • public interface SelectiveStatusListener
    This interface should be used by clients for publishing selective status list changes of its users. 9th January 2008, Guy M. The methods implemented by a SelectiveStatus listerner to receive SelectiveStatus notifications.
