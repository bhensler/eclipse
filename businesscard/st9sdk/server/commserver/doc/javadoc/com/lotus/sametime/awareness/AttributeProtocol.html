        Sends a snapshot of the currentlly available attributes for the the list of specified objects. The snapshot is sent to all sessions that are watching this user. If the user is watched as part of a group then an event is sent to all sessions which are watching the group.
        Parameters:
        watchedObjects - The watched objects.
