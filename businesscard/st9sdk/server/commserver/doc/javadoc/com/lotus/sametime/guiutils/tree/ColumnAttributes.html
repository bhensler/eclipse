      • getRealOrder

        public int getRealOrder(java.util.Vector columns)
        Deprecated. 
        Gets the real order of the columns.
        Parameters:
        columns -
        Returns:
        These columns real order (vs. pseudo order = priority), in a multi column environment. For a column in the left hand side group this will be a negative number, just the same as for the m_pseudoOrder. For the columns in the right hand side group, the column adjecent to (i.e right to) the name column will have realOrder = 1. For the columns in the left hand side group, the column adjecent to (i.e left to) the name column will have realOrder with the highest abs value, i.e the left most column in the group (the farest from name) will have realOrder = -1
