com.ibm.collaboration.realtime.telephony.event
Interface IncomingCallEvent

All Superinterfaces:
CallEvent, java.io.Serializable, TelephonyEvent
public interface IncomingCallEvent
extends CallEvent

IncomingCallEvent is a call event used to signal an incoming call. The listener can, for example, use this event to display a dialog asking the recipient how to handle the incoming call, and then use CallManager.acceptIncomingCall(String, com.ibm.collaboration.realtime.telephony.CallListener) or CallManager.connectIncomingCall(String, com.ibm.collaboration.realtime.telephony.CallOptions, com.ibm.collaboration.realtime.telephony.CallListener) to process the call. TelephonyEvent.getDetails() might contain a user-specified informational message that can be displayed in the UI if appropriate. See CallEvent for more information.

isAudioCall

boolean isAudioCall()
Determines if the incoming call is a computer-to-computer audio call (voice chat), rather than a telephone call. For incoming telephone calls using the Sametime computer phone, this method returns false and isTelephoneCall() returns true. For an incoming computer audio/video call, note that both isAudioCall() and isVideoCall() return true -- these methods are not mutually exclusive.

Returns:
true if the incoming call is a computer audio call, false otherwise.
Since:
Sametime 8.5

setIsConferenceCall

void setIsConferenceCall(boolean flag)
Specifies whether the incoming call is a conference call with more than two participants. This reflects the number of people invited to the call, not necessarily the number of people who actually join the call. This information is provided as a hint to the user interface, but is not required for proper handling of the call. Since the service provider might be unable to determine if the call is a conference call, a value of false (the default) does not necessarily mean the call is not a conference call. Any type of call can be a conference call, if supported by the service provider. This method is for service provider use only.

Parameters:
flag - true if the incoming call is a conference call, false (the default) otherwise
Since:
Sametime 8.0
