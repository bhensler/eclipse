com.ibm.collaboration.realtime.chatwindow.addon
Interface PopupAddOn

public interface PopupAddOn

An interface for adding a popup area beneath the message input are in a chat window. An example of a popup is the file transfer status panel. Multiple popups can be added, in which case tabs are displayed to allow the user to select one of the popups.
