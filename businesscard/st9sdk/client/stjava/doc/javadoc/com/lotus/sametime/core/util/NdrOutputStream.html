Class NdrOutputStream

    public class NdrOutputStream
extends java.io.DataOutputStream
    Our own Mark-able, NDR-compliant data output stream. This stream class writes it's output to a byte array, which can be obtained by calling the method 'toByteArray'.

      • writeInetAddress

        public void writeInetAddress(java.net.InetAddress ip,
                    boolean useNgIpFormat)
                      throws java.io.IOException
        Write an InetAddress to the stream, in "New Generation" format if specified. The "New Generation" format supports IPv4 and IPv6 and is defined as follows: 1 byte: address type (this can be 1 for ipv4 or 2 for ipv6; 0 can specify the invalid type) Then a number of bytes for the address according to the type: For ipv4, 4 bytes For ipv6, 16 bytes For type 0, 0 bytes The highest order byte of the address is dumped first.
        Parameters:
        ip - The address to write.
        useNgIpFormat - Flag indicating whether to use the new IPv6 compatible InetAddress message format.
        Throws:
        java.io.IOException - if something went wrong during writing.
