Package com.lotus.sametime.buddylist.xml

  • Interface Summary 
    Interface Description
    XmlBuddyList
    XmlBuddyList represents a instant messaging contact ("buddy") list.
    XmlBuddyListListener
    XmlBuddyListListener is used to process events in the buddy list APIs
    XmlBuddyListService
    XmlBuddyListService is used to manage an instant messaging contact ("buddy") list.
    XmlGroup
    XmlGroup represents a group in a contact list.
    XmlNestableGroup
    XmlNestableGroup represents a contact list group that can contain other groups, for example, a private group.
    XmlPerson
    XmlPerson represents a person in a contact list.
    XmlPrivateGroup
    XmlPrivateGroup represents a private group in the contact list.
    XmlPublicGroup
    XmlPublicGroup represents a public group in the contact list.
  • Class Summary 
    Class Description
    XmlBuddyListEvent
    XmlBuddyListEvent is the base class for events used by the buddy list APIs.
    XmlCommunity
    XmlCommunity represents a community in a stored contact list.
    XmlConnection
    XmlConnection represents a connection context within an XmlCommunity object.
