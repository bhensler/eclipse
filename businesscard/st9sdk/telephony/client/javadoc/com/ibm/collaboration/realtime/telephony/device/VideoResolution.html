Interface VideoResolution

Deprecated. For internal use only starting with Sametime 8.5.2. 

@Deprecated
public interface VideoResolution

VideoResolution represents the resolution of a video device. Although this interface has been deprecated for Sametime 8.5.2, you can get resolution and other video data for a video call using the new Call.getVideoData() method.

Since:
Sametime 7.5.1
