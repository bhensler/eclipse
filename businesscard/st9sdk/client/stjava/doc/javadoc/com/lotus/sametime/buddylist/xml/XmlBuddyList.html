      • getAllPersons

        XmlPerson[] getAllPersons(boolean uniqueOnly)
        Returns an array of all people contained in this contact list, searching recursively through subgroups. If uniqueOnly is set to true, one person will be added to the returned array, regardless of how many places they appear in the buddylist.

        Note: This method does NOT cycle through public groups, it only goes through private groups defined in the buddylist. To get the contents of a public group, use LookupService.createGroupContentGetter()
        Parameters:
        uniqueOnly - true to only add unique persons to the list, false otherwise.
        Returns:
        Array of people contained in this contact list, null if none.
