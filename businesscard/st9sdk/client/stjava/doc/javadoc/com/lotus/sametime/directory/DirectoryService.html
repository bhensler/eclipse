com.lotus.sametime.directory

Interface DirectoryService

  • All Superinterfaces:
    STCompApi
    All Known Implementing Classes:
    DirectoryComp

    public interface DirectoryService
extends STCompApi
    Defines the directory service used for browsing the Sametime directory. Note that directory browsing may not be available in certain Sametime server configurations. See the toolkit developers guide for details on how to determine if a server supports directory browsing.
