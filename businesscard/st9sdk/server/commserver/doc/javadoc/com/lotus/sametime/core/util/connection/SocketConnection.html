com.lotus.sametime.core.util.connection

Class SocketConnection

  • java.lang.Object
    • com.lotus.sametime.core.util.connection.Connection
      • com.lotus.sametime.core.util.connection.SocketConnection
  • Direct Known Subclasses:
    CustomSocketConnection, DirectSocketConnection, JSSESocketConnection, TLSSocketConnection

    public class SocketConnection
extends Connection
    A plain socket connection. Note that, on certain VMs, this socket connection could be configured to go through a proxy without our knowledge. If you want a socket connection that connects directly, use DirectSocketConnection.
