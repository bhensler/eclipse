                   byte[] value)
        Add or change a our capability.
        Parameters:
        capabilityId - The id of the capability to set.
        value - The value of the Capability. This is a generalized format and the actual values for a capability will depend on the type of capability. The client's role is to know the actual format and to parse the value from byte[].

      • subscribeToUserCapability

        void subscribeToUserCapability(STUser user,
                             int capabilityId)
        Subscribe for a specific capability of a specific user. User subscriptions will receive CapabilityEvent events with getGroup() = null.
        Parameters:
        user - The STUser whose Capabilities we wish to get.
        capabilityId - The capability id for the Capability the client wishes to get. Subscribed capabilities are gotten in the CapabilityEvent.CAPABILITY_CHANGED event.

        Parameters:

      • subscribeToGroupCapability

        void subscribeToGroupCapability(STGroup group,
                              int capabilityId)
        Subscribe to a specific capability of a specific group to get CAPABILITY_CHANGED CapabilityEvents for the STUsers in the group.

        The CapabilityEvent from a user in a subscribed group will have getGroup() = STGroup.

        Parameters:
        group - The STGroup we are interested in getting capability changed events on. We will get CapabilityEvents on all the users in the group who have Capabilities with capability id.
        capabilityId - The id of the Capability we are interested in.

      • unSubscribeFromUserCapability

        void unSubscribeFromUserCapability(STUser user,
                                 int capabilityId)
        Unsubscribe from a specific capability of a specific user who is individually subscribed to stop getting user CapabilityEvents

        Group Capability events will still be received for this user if the user is in a group subscribed to capabilityId.

        Parameters:
        user - The user to unsubscribe.
        capabilityId - The capability id to unsubscribe for the user.

      • unSubscribeFromUser

        void unSubscribeFromUser(STUser user)
        Unsubscribe from a user to stop receiving user CapabilityEvent events for the user. If the user is in a subscribed group, group CapabilityEvents will still be received for the user with CapabilityEvent.getGroup() = STGroup.
        Parameters:
        user - The STUser to unsubscribe from.
