Interface TableModel

    public interface TableModel
    The TableModel interface specifies the methods the Table will use to interrogate a tabular data model. The Table can be set up to display any data model which implements the TableModel interface.

        int getColumnCount()

      • getRowCount

        int getRowCount()
        Deprecated. 
        Returns the number of records managed by the data source object. A Table uses this method to determine how many rows it should create and display. This method should be quick, as it is call by Table quite frequently.
