Class ChatLine

    • com.lotus.sametime.guiutils.chat.ChatLine

    public class ChatLine
extends java.lang.Object
    This is the object representing a line in a ChatTranscript paragraph. The user name (hanging indent) starting each paragraph is a line by itself.
