Note: Implementations of this method should generate both an EVENT_CALL_UNMUTED event for the call and an EVENT_PARTICIPANT_UNMUTED event for each participant.

Parameters:

Throws:

UnknownCallException - if there's no call with the specified ID
UnsupportedCapabilityException - if this operation is not supported
