    public class KeyAction
extends java.lang.Object
    KeyAction This class contains a key code and a method to be executed when the key is pressed. It is used by the KeyHandler to implement mnemonics (Alt+key) in dialogs. See KeyHandler for more details. author : Shiri Lupovich, Feb 2002.
