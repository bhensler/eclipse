Interface SelectiveStatusService

  • All Superinterfaces:
    STCompApi
    All Known Implementing Classes:
    SelectiveStatusComp

    public interface SelectiveStatusService
extends STCompApi
    Definition of the SelectiveStatusService API.

    The SelectiveStatusService provides the facility for a user to cause her default status to trigger the display of a different status for a list of users or groups.
