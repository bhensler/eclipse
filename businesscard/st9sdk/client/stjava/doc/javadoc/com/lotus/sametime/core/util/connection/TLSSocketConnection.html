com.lotus.sametime.core.util.connection

Class TLSSocketConnection

  • java.lang.Object
    • com.lotus.sametime.core.util.connection.Connection
      • com.lotus.sametime.core.util.connection.SocketConnection
        • com.lotus.sametime.core.util.connection.TLSSocketConnection

  • public class TLSSocketConnection
extends SocketConnection
    A direct socket connection. When we try to open a socket, some VMs, notably MSIE, can redirect the connection through a proxy. Sometimes, this is not an accepted behavior. This connection tries to connect directly, bypassing this behavior. Only trusted code (applications and signed applets) can use this connection.
