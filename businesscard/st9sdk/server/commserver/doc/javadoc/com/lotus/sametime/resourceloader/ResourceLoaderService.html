com.lotus.sametime.resourceloader

Interface ResourceLoaderService

  • All Known Implementing Classes:
    ResourceLoaderComp

    public interface ResourceLoaderService
    Interface for loading resources and images in a location independent manner. 4 basic capabilities: 1. Set default search path URL. 2. Get resource bundle from the default path, class ClassLoader or a one time URL. 3. Get an image from the default path, class ClassLoader or a one time URL.

      • getResourceArchive

        boolean getResourceArchive(java.lang.String basename,
                         java.util.Locale locale,
                         java.lang.String codebase)
        This method gets archives of resources, and stores the resources in a local table, so future calls to get them will work. The archives are fetched like resource bundles are: first we look for archive_language_country, then for archive_language, and finally for archive with no suffix. If locale is null, the default locale is used.
        Parameters:
        name - This is base name of the archive.
        locale - The locale to look for. If this is null, only an exact match is looked for.
        codebase - The URL to get the archive from. If this is null, use the default classloader, with it's default parameters.
        Returns:
        true if the archive was found and loaded, false otherwise.
