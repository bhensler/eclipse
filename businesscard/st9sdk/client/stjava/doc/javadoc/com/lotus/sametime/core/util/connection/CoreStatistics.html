Class CoreStatistics

  • java.lang.Object
    • com.lotus.sametime.core.util.connection.CoreStatistics
  • All Implemented Interfaces:
    TimerListener

    public class CoreStatistics
extends java.lang.Object
implements TimerListener
    This class holds the statistics counters for the toolkit core. It has a timer that trace, to a dedicated CSV file, all counters every fix time interval. we use a dedicated timer and not the existing toolkit threads for tracing - as we want to have a counting during a known time interval. By doing this we can calculate rates of changing counters for example in the RC2Sender queues we learn the rate of incoming messages. we use separate trace file and using logger file as we want to have a CSV file that easy to load into data sheet applications for investigation. The name of the trace file and the delay between tracing is configured in StaticProps class.
