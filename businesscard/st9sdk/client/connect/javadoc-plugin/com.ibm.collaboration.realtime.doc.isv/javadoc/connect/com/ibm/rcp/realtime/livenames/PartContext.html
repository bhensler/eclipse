com.ibm.rcp.realtime.livenames
Interface PartContext

public interface PartContext

These CONTEXT_xxx constants serve as a reference to be used by livename objectContributions in the plugin.xml, to control what contexts they appear in.

For example you can filter actions using the eclipse "filter" element using the name "context" or "notContext" and the value from these interfaces. For example:

An objectContribution with the following filter will ONLY show up in the context menu shown by the contact list:

<filter name="context" value="ChatWindow"/>

An objectContribution with the following filter will show up everywhere EXCEPT in the context menu shown by the business card livename:

<filter name="notContext" value="BizCard"/>
