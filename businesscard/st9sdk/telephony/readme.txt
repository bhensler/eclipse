                      IBM Sametime Telephony Toolkit

This readme provides important information about the IBM Sametime Telephony
Toolkit, which is part of the IBM Sametime Software Development Kit (SDK).

The IBM Sametime Telephony Toolkit provides documentation and samples to enable
service providers to integrate telephony services, including voice and video, with
IBM Sametime. Telephone calling services can be provided for either the IBM
Sametime Connect client, the Sametime server, or both. For the Sametime Connect
client only, service providers can also integrate voice over IP (VoIP) and video
services using the APIs included in this toolkit.


-----------------------------------------
IBM SAMETIME TELEPHONY TOOLKIT CONTENTS
-----------------------------------------

The IBM Sametime Telephony Toolkit directory structure is shown below (other
contents of the Sametime SDK are not shown):

st900sdk
   telephony
      readme.txt                             - this readme file
      client                                 - Client Telephony API Toolkit
         doc
            API_Changes.txt                  - List of API changes from 8.5
            ClientTelephony.pdf              - Client Telephony API Guide
         javadoc                             - Client Telephony API Javadoc(TM)
         samples                             - Client Telephony API sample plug-ins
      tcspi                                  - TCSPI Toolkit
    
See the file readme.txt in the tcspi directory for more information about the
contents of the TCSPI Toolkit.
