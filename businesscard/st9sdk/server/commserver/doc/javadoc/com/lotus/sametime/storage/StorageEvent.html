    • Method Detail

      • getRequestResult

        public int getRequestResult()
        Gets the request code result. If the request was a query the possible result codes are the following:
        ST_OK - All requested attributes were retrieved successfully.
        ST_FAIL - None of the requested attributes were retrieved.
        STORAGE_ATTRIB_NOT_EXIST - Some of the requested attributes were not retrieved because they have never been stored on the Sametime server.
        If the request was a store request the possible result codes are the following:
        ST_OK - All requested attributes were stored successfully.
        St_Fail - None of the requested attributes were stored.
        Returns:
        The request result code.
        See Also:
        STError

      • getAttrList

        public java.util.Vector getAttrList()
        For a query request, it is the list of successfully retrieved attributes. For a store request, it is the list of storage attributes that were stored successfully or not as interpreted by the value of getRequestResult():
        ST_OK - The list of attributes stored successfully.
        ST_FAIL - The list of attributes that were not stored successfully.
        Returns:
        The list of attributes that are associated with this event.
        See Also:
        STError
