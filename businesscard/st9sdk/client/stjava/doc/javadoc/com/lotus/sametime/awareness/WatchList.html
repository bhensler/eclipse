com.lotus.sametime.awareness

Class WatchList

  • java.lang.Object
    • com.lotus.sametime.awareness.WatchList

  • public class WatchList
extends java.lang.Object
    Represents a list of watched Sametime objects (users or groups). Provides methods to add and remove objects from the list and notifications about changes in state of the watched objects.
    See Also:
    AwarenessService.addToAttrFilter(int[]), AwarenessService.addToAttrFilter(int[])
