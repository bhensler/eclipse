public class AlertUserActionMessage
extends Message

AlertUserActionMessage is used in conjunction with AlertButton, and is generated when a user clicks a button on a pop-up alert. Note that AlertUserActionMessage is not used for alerts generated internally by Sametime Connect, so you can only use this message to handle alerts that you've created using AlertUserMessage. Use Message.setSourceId(String) to specify a unique identifier for your plug-in, for example, plug-in ID or class name, so that you can distinguish messages generated by your alerts from those generated by other third-party plug-ins.
