Class AwarenessPermissionManager

  • java.lang.Object
    • com.lotus.sametime.community.AwarenessPermissionManager
  • All Implemented Interfaces:
    MyPrivacyListener, java.util.EventListener

    public class AwarenessPermissionManager
extends java.lang.Object
implements MyPrivacyListener
    Represents an object to handle awareness permission request and approval.

    Note: The sametime.ini flag ST_DEFAULT_PRIVACY under [Config] determines whether privacy settings of a user default to "everybody can see me" or "no one can see me." The default value of 0 means that "everyone can see me," and should not be changed for conventional installations. Installations that require the awareness permission feature should set this flag to 1, to indicate that users who never touched their privacy settings are initially not visible to anyone.
