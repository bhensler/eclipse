      • m_putAllMessagesInDispatcherQueue

        public static boolean m_putAllMessagesInDispatcherQueue
        This flag enables the application to choose whether to use the dispatcher queue for incoming and outgoing messages or only for incoming messages from the server. By default every message is put in the dispatcher queue and handled in turn. If the message holds a request to the server , then when it is retrieved from the dispatcher queue, a relevant stream is built and sent on the channel. When this flag is set to false, messages that contain requests for the server , will be put on the sender queue directly.
