Class HybridPollingConnection

  • java.lang.Object
    • com.lotus.sametime.core.util.connection.Connection
      • com.lotus.sametime.core.util.connection.HybridPollingConnection

  • public class HybridPollingConnection
extends Connection
    An HTTP-tunneling connection using the Hybrid Polling method. This class can connect using either the built-in URLConnection class provided by the JVM or the native connection code provided by the sturlcon10 library, which uses MS Windows WinInet API for connectivity, on MS Windows machines. Connection using the JVM's built-in URLConnection class requires the Java plugin. Connection using the sturlcon10 library will require installation of the sturlcon10.dll file on the client machine, and giving applets the right permissions to load and access this lib. The sturlcon10 lib is capable of accessing and using the Internet Explorer proxy settings and certificate store. This class deprecates the AgentConnection class for a Java 2 environment. .
