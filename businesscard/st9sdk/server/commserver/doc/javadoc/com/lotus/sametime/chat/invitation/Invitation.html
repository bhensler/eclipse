com.lotus.sametime.chat.invitation

Class Invitation

  • java.lang.Object
    • com.lotus.sametime.chat.invitation.Invitation

  • public class Invitation
extends java.lang.Object
    Class representing an invitation to a Sametime meeting.

    Note: These Invitation APIs are focused on N-Way chats, not audio, video, application sharing, calls, etc. So the meeting type should primarily be: MeetingTypes.ST_CHAT_MEETING
    Since:
    Sametime 8.5.1
