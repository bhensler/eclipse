clearEmbeddedObjects

void clearEmbeddedObjects(boolean run)
Removes any embeddedable objects that were added to the message input area using embedObject(Embeddable), and optionally executes the run method in each of those objects. For example, when the user drags a file to the message input area, the file transfer plug-in embeds a file icon there; when the user presses Send, the file transfer plug-in removes the embedded icon and uses the run method to begin the file transfer.

Parameters:
run - true to execute the embedded object's run method, false otherwise

embedObject

void embedObject(Embeddable embeddable)
Embeds an object in the message input area. For example, when the user drags a file to the message input area, the file transfer plug-in embeds a file icon there. You can embed more than one object by calling embedObject multiple times. To remove an embedded object, use clearEmbeddedObjects(boolean). Note that there is no way to remove just one of many embedded objects.

Parameters:
embeddable - Embeddable object
