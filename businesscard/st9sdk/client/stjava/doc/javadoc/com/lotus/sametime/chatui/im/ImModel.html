Interface ImModel

  • All Known Implementing Classes:

    Deprecated. 
    due to the availability of additional non-AWT GUI frameworks. This component is no longer supported.

    public interface ImModel
    The interface of the model for the IM panel. Provides methods for sending messages and reciving notifications for incoming messages and events.
