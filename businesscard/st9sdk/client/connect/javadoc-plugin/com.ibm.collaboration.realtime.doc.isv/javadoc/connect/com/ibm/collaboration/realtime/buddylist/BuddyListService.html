getBuddyList

BuddyList getBuddyList()
Returns the current "live" buddylist. Actions taken on the groups in the buddylist will show up in the UI immediately. The BuddyList does not need to be reset. New groups that are added or removed will appear in the UI in realtime.

resetBuddyList

void resetBuddyList(BuddyList newBuddyList,
                    boolean save)
Replaces the current buddylist with the given buddylist. This should only be called if replacing the buddylist with a completely different BuddyList (which should rarely if ever be done), or if new root groups have been added to an existing BuddyList. The buddylist does not need to be reset if removing groups and persons from other groups since those actions take effect in the UI immediately, though the buddylist does need to be saved after such actions.
