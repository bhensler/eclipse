Class TokenServiceAdapter

  • java.lang.Object
    • com.lotus.sametime.token.TokenServiceAdapter
  • All Implemented Interfaces:
    TokenServiceListener, TokenServiceListener2, java.util.EventListener

    public class TokenServiceAdapter
extends java.lang.Object
implements TokenServiceListener, TokenServiceListener2
    An abstract adapter class for receiving token service events. TokenServiceAdapter is extended by classes that implement the TokenServiceListener or TokenServiceListener2 interfaces. The use of this base adatper class is recommended so that the interface(s) can be evolved in future releases without breaking existing implementations.
