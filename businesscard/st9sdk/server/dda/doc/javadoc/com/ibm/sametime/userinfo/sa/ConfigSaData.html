Class ConfigSaData

  • java.lang.Object
    • com.ibm.sametime.userinfo.sa.ConfigSaData
  • All Implemented Interfaces:
    com.lotus.sametime.configuration.ConfigServiceListener

    public class ConfigSaData
extends java.lang.Object
implements com.lotus.sametime.configuration.ConfigServiceListener
    This class manages configuration updates received from the ST Configuration service. It stores the received data and combines it with the data read from UserInfoConfig.xml. The data irems read from ST Configueration overrides similar exixsting data items in UserInfoConfig.xml The updates being received are Ldap upfdates and UserInfo updates. The updates are only received once when the application loads. In order to refresh the configuration data the Userinfo service should be restarted.
