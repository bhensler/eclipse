Package com.ibm.collaboration.realtime.telephony.exception

Exception Summary
InactiveCallException InactiveCallException is thrown when an operation is attempted on a call that is no longer active.
InvalidConfigurationException InvalidConfigurationException is thrown when accessing a service provider or conference configuration or that is no longer valid.
InvalidOperationException InvalidOperationException is thrown when an application attempts to perform an operation that is not supported, for example, changing the call ID for an existing call.
InvalidParameterException InvalidParameterException is thrown when in invalid or null parameter is passed to a client-side telephony API method.
ServiceUnavailableException ServiceUnavailableException is thrown when a requested telephone service is unavailable.
TelephonyException TelephonyException is the base class for all exception classes in the client-side telephony API.
TelephonyRuntimeException TelephonyRuntimeException is the base class for all unchecked exception classes in the client-side telephony API.
TelephonyServiceException TelephonyServiceException is used for service exceptions generated by the telephony service provider.
UnknownAddressException UnknownAddressException is thrown when an invalid or unknown address is specified when attempting to call a user.
UnknownCallException UnknownCallException is thrown when an invalid or unknown call ID is specified to a client telephony service API method.
UnknownParticipantException UnknownParticipantException is thrown when an invalid or unknown participant ID is specified to a client telephony service API method.
UnsupportedCapabilityException UnsupportedCapabilityException is thrown when an application attempts to perform an operation that is not supported by the telephony service provider.
