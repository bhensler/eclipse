    Deprecated. 
    due to the availability of additional non-AWT GUI frameworks. This component is no longer supported.

    public class DirectoryPanel
extends java.awt.Panel
implements ContentProvider
    The panel provides a view of the available directories and their contents. Enables performing the following operations: Select a specific directory from the list of available directories, browse and search for users and groups within a directory, and extract the selected user and group information.
