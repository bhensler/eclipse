Interface TelephonyAdapterService

  • All Known Implementing Classes:
    TelephonyAdapterComp

    public interface TelephonyAdapterService
    Telephony Presence Adapter Services interface. Defines the provided services: Publish telephony status. Subscribe for Sametime status changes. Unsubscribe. Get Sametime status changes notifications.

      • initiate

        void initiate(java.util.Properties taProperties,
            java.util.Properties mappingProperties,
            java.util.Properties crProperties)
        Perform initialization steps to the Sametime community and to this system's component in order to be able to handle requests The proevided properties are used. The configuration file content is ignored. Using this initiation method skip the load of properties from the configuration files of both the TelephonyAdapter and the CachedResolve components.
        Parameters:
        taProperties - - the TelephonyAdapter initialization properties. The list of properties is defined at the st.telephony.adapter.properties file.
        mappingProperties - - the TelephonyAdapter mapping properties. The list of properties is defined at the override.status.to.action.mapping file.
        crProperties - - the CachedResolve initialization properties. The list of properties is defined at the st.cached.resolve.properties file.
