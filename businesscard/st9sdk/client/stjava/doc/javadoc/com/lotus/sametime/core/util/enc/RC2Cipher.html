com.lotus.sametime.core.util.enc

Class RC2Cipher

  • java.lang.Object
    • com.lotus.sametime.core.util.enc.RC2Cipher

  • public class RC2Cipher
extends java.lang.Object
    This class is modelled after the Cipher class found in JCE. The methods provided here are a subset of that class. Note that this class currently only provides support for the RC2 encryption algorithm, in CBC mode, using PKCS #5 padding.
