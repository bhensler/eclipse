com.ibm.sametime.userinfo.userInfobbapi

Class Response

  • java.lang.Object
    • com.ibm.sametime.userinfo.userInfobbapi.Response

  • public class Response
extends java.lang.Object
    An object that holds data retrieved from a data repository. It contains a map with the detail's properties and retrieved values. The key of this map is a string that contains the name of the detail and the value is a DetailItem object.

        public boolean isUserFound()
        Verifies whether data has been found for the relevant request, even if this data includes only an empty entry containing the user ID. The value returned by this method can be true if some of the requested attributes were found for this user or if the user is found in the storage but there is no available data for any of the requested details.
        Returns:
        boolean True if the user exists
