public class AdminRoomServices
extends RoomServices

This sub-class of RoomServices provides "super" user operations such as creating and managing other users' rooms. If administrator services are required, this class should be used. A valid administrator user name/ password is required. See also RoomServices

createRoomOnBehalfOf

public MeetingRoom createRoomOnBehalfOf(String name,
                                        String password,
                                        String ownerId)
                                 throws RoomException
Allows an administrator to create a room on behalf of the specified user with the provided name and password. The caller will have to be a user with administrative privileges on the server. The newly created room will be entirely owned and managed by the end user. The room is created with a version 4 GUID uniquely identifying the room. An ID is automatically generated for each room at time of creation.

Parameters:
name - If the room name is empty or null, a RoomException will be thrown.
password - Room password
ownerId - userId ID of the user on whose behalf the room is being created.
Returns:
MeetingRoom Returns the newly created meeting room.
Throws:
RoomException
