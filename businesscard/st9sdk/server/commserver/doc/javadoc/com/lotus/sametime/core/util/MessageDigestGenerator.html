Class MessageDigestGenerator

  • java.lang.Object
    • com.lotus.sametime.core.util.MessageDigestGenerator

  • public class MessageDigestGenerator
extends java.lang.Object
    Generates MessageDigest objects without throwing exceptions. Provides a global MessageDigest object that is pre-initialized with a (global) "salt" value.
