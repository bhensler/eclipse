sendAnnouncement

void sendAnnouncement(java.lang.String message,
                      Person[] recipients,
                      boolean allowResponse)
Send an announcement message to the list of recipients specified. NOTE: This API is not intended to be used for sending announcements to a massive list of recipients (1000s). The API uses Person objects which require presence subscriptions so using this API to send to a very large group of recipients would have performance implications. Please see the com.lotus.sametime.announcement.AnnouncementService in the Sametime Java Toolkit as an alternative in that case.

Parameters:
message - The announcement message to send
recipients - The users that will receive the message.
allowResponse - Indicates whether the receipent is expected/allowed
