lookup

void lookup(java.lang.String name)
Looks up the specified user or group name in the directory. The name can be specified in any format supported by the underlying directory search implementation. Before calling this method,use addLookupListener(LookupListener) to register a lookup event listener. Information about the specified user or group will be sent to that listener in a LookupEvent object; use LookupEvent.getContactInfos() or LookupEvent.getGroupInfos() to access the user or group information. The there is more than one possible match for the specified name, multiple results may be returned. To limit the number of results returned, use setMaxResults(int).

Parameters:
name - user or group name
