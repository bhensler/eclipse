com.ibm.rtc.client.rtc4web
Interface DispatchAdapter

public interface DispatchAdapter

An optional class, used by an application to adapt an update before it is dispatched, for example to perform a thread switch to call RealTimeSession.dispatchUpdate(ArrayList) on another thread. If this class is not set on the RealTimeSession, then updates are delivered in their complete form, on the RealTimeSession's update thread.
