Interface DirectoryInfo

public interface DirectoryInfo

DirectoryInfo provides access to a user's directory information. The information is represented as a collection of attribute key/value pairs. The constants defined in this interface define the supported attribute key names. All the attribute values are String unless otherwise noted.
