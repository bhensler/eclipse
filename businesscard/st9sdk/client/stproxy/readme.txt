This readme provides important information about the IBM Lotus Sametime Browser
Client Toolkit, which is part of the IBM Lotus Sametime Software Development Kit (SDK).

The IBM Lotus Sametime Browser Client Toolkit is a collection of APIs that can be used
to build web enabled communication enabled business (CEBP) processes by embedding
presence, instant messaging and other real-time capabilities inside a web application.

Additional information about Sametime is available at http://www.lotus.com/sametime

-----------------------------------------------
LOTUS SAMETIME BROWSER CLIENT TOOLKIT CONTENTS
-----------------------------------------------

The IBM Lotus Sametime Browser Client Toolkit provides documentation and samples to
help you build web pages that are enabled with Sametime capabilities.

The Lotus Sametime Browser Client Toolkit includes the following components.

- IBM Lotus Sametime Browser Client Toolkit Developer's Guide. This document
  describes the web programming model for interfacing with Lotus Sametime Proxy Server 9
  and later releases, and provides the information you need to build your own Sametime-
  enabled web pages.

- Sample web pages. The toolkit samples consist of several HTML pages that demonstrate different
  levels of the functionality available in the toolkit.

The Lotus Sametime Browser Client Toolkit directory structure is shown below
(other contents of the Lotus Sametime SDK are not shown):

st9sdk
   client
      stproxy
         readme.txt                          - this readme file
         doc
            STBrowserSDK.pdf  	             - Lotus Sametime Browser Client Toolkit Developer's Guide
         samples                             - Sample HTML pages
