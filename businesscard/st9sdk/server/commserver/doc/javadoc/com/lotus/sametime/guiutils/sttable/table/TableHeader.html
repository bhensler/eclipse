Class TableHeader

  • java.lang.Object

      • com.lotus.sametime.guiutils.sttable.table.TableHeader

    Deprecated. 
    due to the availability of additional non-AWT frameworks. This AWT component is no longer supported.

    public class TableHeader
extends java.awt.Component
    This is the column header part of a Table. It allows the user to change column widths. Rendering options can be changed by calling corresponding setters:
     setBackground(Color c)
    - to set background color;
     setForeground(Color c)
    - to set foreground color;
     setFont(Font f)
    - to set header font;
    To obtain instance of header, call 
     table.getTableHeader()
    .
