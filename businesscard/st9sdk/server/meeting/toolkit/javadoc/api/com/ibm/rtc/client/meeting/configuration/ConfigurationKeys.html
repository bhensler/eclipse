RTC4WEB_ENFORCE_NONCE

static final String RTC4WEB_ENFORCE_NONCE
rtc4web.enforceNonce=Determines whether the server will allow service only if a "nonce" exists in the HTTP requests originating from a client. The nonce is a per user, per session GUID which is issued once, when the user joins the session, and is removed when the user leaves the session. Enforcing the nonce existence on each HTTP request acts as a counter measure against common HTTP and browser-based vulnerabilities, such as XSS and CSRF attacks. The default is true.

Values:

  true
  false

See Also:

USERINFO_URL_TEMPLATE

static final String USERINFO_URL_TEMPLATE
userInfoUrlTemplate=This key is required if userInfoRedirect is true. Enter the external URL where user photos are stored. The URL must contain "{0}". As user images are fetched, the Sametime Meeting Server will replace {0}, with an e-mail address.

Example:

  If you enter this external URL: http://images.company.com/users/{0}.jpg

  Then an image for user1 is retrieved from this URL: http://images.company.com/users/user1@company.com.jpg
