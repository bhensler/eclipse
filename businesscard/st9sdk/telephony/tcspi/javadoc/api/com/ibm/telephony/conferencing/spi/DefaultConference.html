registerUser

public void registerUser(java.lang.String userName,
                         java.lang.String userRole,
                         java.lang.String userPin,
                         ConferenceResponseHandler listener)
Description copied from interface: Conference
This method is a required call to setup the service provider for all subsequent actions for users. The user name is used to identify a user as a resource user and will return a unique user object. This user can then be used to provide user based conference control. For any single user name registerUser may be called multiple times with out affecting state. However, once the initial role is set it is expected that User.setProperty to change a users role after registration. Successful request: listener.registerUser is called Event contents: original name, role, and a User object scoped to the conference. Failed Request: listener.requestFailed with appropriate error code Possible error codes:

userName - uniquely identifies the user.
