Class AlertSettingsData

java.lang.Object
  com.ibm.rcp.realtime.alerts.AlertSettingsData

public class AlertSettingsData
extends java.lang.Object

AlertSettingsData encapsulates all the settings details for alert preferences. AlertSettingsData can be used when extending DefaultAlertWindow.
