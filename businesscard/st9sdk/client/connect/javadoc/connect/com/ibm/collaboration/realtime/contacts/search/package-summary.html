Package com.ibm.collaboration.realtime.contacts.search

Interface Summary
ContactInfo ContactInfo represents an instant messaging user contact.
DirectoryInfo DirectoryInfo provides access to a user's directory information.
DirectoryService DirectoryService provides methods to obtain directory information about users.
DirectoryServiceFactory DirectoryServiceFactory provides access to a DirectoryService implementation instance for a given community.
GroupContentLookup GroupContentLookup is used search for members of a group.
GroupInfo GroupInfo represents an instant messaging group.
GroupLookupListener GroupLookupListener is used to process GroupContentLookupEvent objects generated by GroupContentLookup.queryGroupContent(String).
Lookup Lookup provides methods to look up user or groups in the directory.
LookupListener LookupListener is used to process LookupEvent objects generated by Lookup.lookup(String) or Lookup.lookup(String[]).
Node Node is a base interface representing an entry in a instant messaging contact list.
UserLookupListener UserLookupListener is used to process UserInfoLookupEvent objects generated by DirectoryService.queryUserAttributes(String, String[]).
 

Class Summary
GroupContentLookupEvent GroupContentLookupEvent is used to return group information requested using GroupContentLookup.queryGroupContent(String).
LookupEvent LookupEvent is used to return user or group information requested using Lookup.lookup(String) or Lookup.lookup(String[]).
UserInfoLookupEvent UserInfoLookupEvent is used to return user information requested using DirectoryService.queryUserAttributes(String, String[]).
