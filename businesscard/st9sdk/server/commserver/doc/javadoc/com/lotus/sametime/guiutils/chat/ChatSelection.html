Class ChatSelection

  • java.lang.Object
    • com.lotus.sametime.guiutils.chat.ChatSelection
  • Deprecated. 
    due to the availability of additional non-AWT frameworks. This AWT component is no longer supported.

    public class ChatSelection
extends java.lang.Object
    This is a helper object for ChatSelector. ChatSelection describes a selection point in the transcript's text defined by a paragraph index and a zero relative offset in the paragraph.
