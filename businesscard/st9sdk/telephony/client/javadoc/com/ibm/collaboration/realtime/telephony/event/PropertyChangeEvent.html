Interface PropertyChangeEvent

All Superinterfaces:
CallEvent, java.io.Serializable, TelephonyEvent
public interface PropertyChangeEvent
extends CallEvent

PropertyChangeEvent is a call event used to signal a change to the properties of the associated call or one of the call participants. See CallEvent for more information.
