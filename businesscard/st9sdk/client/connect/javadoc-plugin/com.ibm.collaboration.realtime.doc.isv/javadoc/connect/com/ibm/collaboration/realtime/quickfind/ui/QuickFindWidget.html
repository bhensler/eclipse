public abstract class QuickFindWidget
extends org.eclipse.swt.widgets.Composite

QuickFindWidget is a reusable widget that allows you to easily add quick find capabilities to any window in Sametime. The QuickFindWidget includes a text box to enter the name to search on, a search button to initiate the search, a drop-down containing results, and several hooks you can implement to control the functionality.

Since:
Sametime 8.0
