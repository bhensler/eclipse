    public class NdrInputStream
extends java.io.DataInputStream
    Our own Mark-able, NDR-compliant data input stream. This stream class reads it's NDR-complient input from a byte array, which is given to it on construction.

      • readInetAddress

        public java.net.InetAddress readInetAddress(boolean useNgIpFormat)
                                     throws java.io.IOException
        Read an InetAddress to the stream, in "New Generation" format if specified. The "New Generation" format supports IPv4 and IPv6 and is defined as follows: 1 byte: address type (this can be 1 for ipv4 or 2 for ipv6; 0 can specify the invalid type) Then a number of bytes for the address according to the type: For ipv4, 4 bytes For ipv6, 16 bytes For type 0, 0 bytes The highest order byte of the address is read first.
        Parameters:
        useNgIpFormat - Flag indicating whether to use the new IPv6 compatible InetAddress message format.
        Returns:
        The read address.
        Throws:
        java.io.IOException - if something went wrong during reading.
