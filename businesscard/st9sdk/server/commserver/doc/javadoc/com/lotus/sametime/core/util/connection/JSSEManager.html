com.lotus.sametime.core.util.connection

Class JSSEManager

  • java.lang.Object
    • com.lotus.sametime.core.util.connection.JSSEManager

  • public class JSSEManager
extends java.lang.Object
    A core class that manages JVM security settings and JSSE parameters. It collects necessary parameters, such as keystore and credentials, and initializes JSSE providers based on JVM version. If the JVM is from IBM and FIPS mode is turned on (by specifying "-Dcom.ibm.jsse2.JSSEFIPS=true" system * property), it dynamically loads the IBMJCEFIPS provider to perform FIPS-compliant cryptographic functionality. Applications which use JSSESocketConnection to create TLS connections must first call the initJSSE() method and provide required parameters.
    Since:
    Sametime 8.5
