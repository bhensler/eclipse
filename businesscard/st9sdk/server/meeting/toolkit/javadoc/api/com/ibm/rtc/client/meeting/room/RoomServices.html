public class RoomServices
extends ServiceBase

Provides a way to perform various room related operations including creation, editing, deletion, search etc..All operations require logging in to the server as an authentic user.The server URL complete with host name and port should be passed into this class.

Example: 

 RoomServices roomServices = new RoomServices("http://server.mycompany.com:80");
 roomServices.login(userName,password);
A MalformedURLException will be thrown if the server URL is not well-formed. If an invalid host name is entered, an UnknownHostException will be thrown at runtime. Rooms can be created with only a name or a permaname. The owner of the room and creator are managers of the room automatically. Other room properties take on the default values. In general, the default settings are permissive and require user edits to make it otherwise. 
 roomServices.createRoom("myroom");
Rooms can also be created by passing a MeetingRoom object set with specific properties. 
 MeetingRoom room = new MeetingRoom();
 room.setDescription("XYZ");
 room.setIsDisposable(true);
 room.setUnlisted(true);
 roomServices.createRoom("myroom","password",room);
