public class ChatWindowAction
extends org.eclipse.jface.action.Action
implements ChatWindowExtension

An org.eclipse.jface.action.Action that provides functionality to the chat window.

It is recommended that if you want to override any Action methods such as isEnabled() to determine enablement, you use this class instead of just using the ChatActionDelegate interfaces.
