setRebroadcastStatus

void setRebroadcastStatus(boolean rebroadcast)
Sets the rebroadcast status boolean of this resolver. If set to true, the LiveNameResolver will rebroadcast status events for names that have already been resolved by this resolver instance. In other words, each time the caller attempts to resolve a particular name, a statusChange event will occur.

Parameters:
rebroadcast - whether or not to rebroadcast status events for already watched names.
