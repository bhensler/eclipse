    public class UserInfoServlet
extends javax.servlet.http.HttpServlet
    Author:
    gali The servlet class : eccepts requests and forwards them to the Manager recievs requests for user info. The request has 3 data items that the application requires The user id to search information for The set requested -optional The type of operation requested. 3 types are supported 1- a request to retrieve a given list of parameter for a given userid 2 - a request to retrieve the items in a specific predefined set which its number is submitted as parameter. 3 - a request to retrieve the values for the details in a specific given set for a given userid
