Interface PreferenceListExtension

All Known Implementing Classes:
NotifyWindowPreference
public interface PreferenceListExtension

PreferenceListExtension provides the ability to add to the list of preferences in existing preference pages in Sametime that support preference lists.

Since:
Sametime 8.0
