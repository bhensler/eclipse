com.lotus.sametime.storagesa

Interface SAppStorageService

  • All Superinterfaces:
    STCompApi
    All Known Implementing Classes:
    SAppStorageComp

    public interface SAppStorageService
extends STCompApi
    Defines the storage service for a Service Application. Used for storing user related attributes on the Sametime server. This service is different from the client service by allowing the application to access all of the clients data.

      • enableBuffering

        void enableBuffering(boolean on)
        Enables/disables buffering of requests when the storage service on the Sametime server is down. When enabled, all requests sent to the component when the service is down will be saved and sent to the server when it comes back up. The default behavior is buffering disabled.
        Parameters:
        on - Whether or not buffering is enabled. true if enabled; false otherwise.

      • disableDefaultLocate

        void disableDefaultLocate()
