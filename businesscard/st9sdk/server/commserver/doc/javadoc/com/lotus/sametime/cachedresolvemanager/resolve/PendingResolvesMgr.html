Class PendingResolvesMgr

  • java.lang.Object
    • com.lotus.sametime.cachedresolvemanager.resolve.PendingResolvesMgr
  • All Implemented Interfaces:
    TimerListener

    public class PendingResolvesMgr
extends java.lang.Object
implements TimerListener
    This class manages user resolve request are not responded for a long time. It goes over the sorted resolve request list in periodical manner and detects the user requests that are supposed to be resolved. If a resolve request is not responded during a specific estimated time, the PendingResolvesMgr regards it as a resolve in process and notifies the CachedResolveListener accordingly.
