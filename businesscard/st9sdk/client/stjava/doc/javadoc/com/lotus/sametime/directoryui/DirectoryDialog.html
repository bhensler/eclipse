    public class DirectoryDialog
extends java.awt.Dialog
    The dialog provides a view of the available directories and their contents. Enables to perform the following operations: Select a specific directory from the list of available directories, browse and search for users and groups within a directory, extract the selected user and group information. The dialogs contains the directory panel and additiional set of buttons that can be customized. The default dialog also includes the ability to select a view from the directory and view its content.
