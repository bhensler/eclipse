      • getResourceArchive

        public boolean getResourceArchive(java.lang.String basename,
                         java.util.Locale locale,
                         java.lang.String codebase)
        This method gets archives of resources, and stores the resources in a local table, so future calls to get them will work. The archives are fetched like resource bundles are: first we look for archive_language_country, then for archive_language, and finally for archive with no suffix. If locale is null, the default locale is used. The 'get resource using default codebase' will not work in NS when using the default codebase, since NS does not allow getting .zip files using the classloader.
        Specified by:
        getResourceArchive in interface ResourceLoaderService
        Parameters:
        name - This is base name of the archive.
        locale - The locale to look for. If this is null, only an exact match is looked for.
        codebase - The URL to get the archive from. If this is null, use the default classloader, with it's default parameters.
        Returns:
        true if the archive was found and loaded, false otherwise.
