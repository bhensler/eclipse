com.lotus.sametime.cachedresolvemanager.batch

Class CachedResolveBatchMode

  • java.lang.Object
    • com.lotus.sametime.cachedresolvemanager.batch.CachedResolveBatchMode
  • All Implemented Interfaces:
    CachedResolveListener, LoginListener, java.util.EventListener

    public class CachedResolveBatchMode
extends java.lang.Object
implements LoginListener, CachedResolveListener
    Implements CachedResolveService client that works like this: 1. Reads external user names for resolving from the text file. 2. Resolves external user names using resolveUser API of the CachedResolveService component. 3. When users resolving is finished (users are resolved successfully, or some resolves failed, or some resolves are not responded yet), the application invokes flushMemoryToFile API for storage the successfully resolved users data in the persistent file.
