com.lotus.sametime.location

Interface LocationService

  • All Superinterfaces:
    STCompApi
    All Known Implementing Classes:
    LocationComp

    public interface LocationService
extends STCompApi
    Defines the API for the Location Service. This service provides methods for
  • Adding and removing user location represented by the Location object, Adding and removing a Location listener to listen for Location notifications. Adding and removing users to a watch list to receive Location notifications for those users.

        Sets the Location for the user logged into this component's session. If successful, this user, if implementing LocationServiceListener, will receive a myLocationChanged() notification and if not successful, will receive a myLocationFailed() notification. Other users who have this user on their watch lists and who implement LocationServiceListener will receive a locationChanged() notification if the Location was successfully set.
