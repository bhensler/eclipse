public class CallAction
extends org.eclipse.jface.action.Action
implements com.ibm.collaboration.realtime.telephony.ui.extension.ICallUIExtension

Action class for actions that appear on the telephony toolbar area in a telephony or video chat window, using the com.ibm.collaboration.realtime.telephony.ui.callAction extension point. The toolbar area consists of three toolbars, one next to the other.
