public interface CallEvent

CallEvent represents a call-related telephony event, processed by CallListener.processCallEvent(CallEvent).

EVENT_CALL_STARTING

static final java.lang.String EVENT_CALL_STARTING
Generated when a new call is about to start, before the service provider's startCall method is called. This event is generated by Sametime Connect, so service providers should not generate the event. Note that this event is only generated for outgoing calls, not incoming calls - to be notified of incoming calls, use IncomingCallHandler. Note also that the call ID is not yet available when this event is generated.

Sametime 8.0.1
