Class ChatController

  • java.lang.Object
    • com.lotus.sametime.awarenessui.AwarenessViewController

  • Direct Known Subclasses:
    AVController, ConfController
    Deprecated. 
    due to the availability of additional non-AWT GUI frameworks. This component is no longer supported.

    public class ChatController
extends AwarenessViewController
    A controller for the awareness view that provides basic chat capabilities. Enables starting either one on one chat sessions or a conference chat between two or more users.
