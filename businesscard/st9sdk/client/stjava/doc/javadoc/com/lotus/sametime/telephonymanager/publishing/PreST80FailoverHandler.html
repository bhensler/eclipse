Class PreST80FailoverHandler

  • java.lang.Object
    • com.lotus.sametime.telephonymanager.publishing.PreST80FailoverHandler
  • All Implemented Interfaces:
    ClusterManagerListener, UserAttributeSAListener

    public class PreST80FailoverHandler
extends java.lang.Object
implements ClusterManagerListener, UserAttributeSAListener
    Handles the publishing part of failover recovery against a ST community which doesn't support "passive" users The class add itself to the cluster manager as a listener (during initilization) For each logged in server it gets the UserAttribuyeSAService and add itself as listener too When a Sametime server fails, the UserAttributeSAService sends a setAttributeFailed for each user which has an attribute on the failed server We'll create an appropriate event and send it to the impl
