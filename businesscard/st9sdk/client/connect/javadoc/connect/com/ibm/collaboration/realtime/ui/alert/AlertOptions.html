com.ibm.collaboration.realtime.ui.alert
Interface AlertOptions

public interface AlertOptions

AlertOptions provides options used to customize the appearance of a pop-up alert. Use AlertOptions in conjunction with AlertUserMessage to display a pop-up alert.
