com.ibm.collaboration.realtime.orgtree
Interface OrgNode

public interface OrgNode

The OrgNode is an abstraction for an entry in the directory. An OrgNode can represent either a person or a group.

Since:
Sametime 8.5.2 IFR 1

GROUP_TYPE_DISPLAY_CONTAINER

static final java.lang.String GROUP_TYPE_DISPLAY_CONTAINER
A string constant that can be used as the value of the ATTR_GROUP_TYPE. An OrgNode whose group type is displayContainer is shown in the UI as a container of the actual root nodes. It is not a true directory entry, simply a display OrgNode that contains other OrgNodes

ATTR_IS_PARENT

static final java.lang.String ATTR_IS_PARENT
Attrib used to indicate whether or not this node is a parent of other nodes, i.e whether or not it is a container for child nodes. In an attribute based org tree, this would be true if for example the node represents a manager. The children of this node may be empty until they are actually retrieved from the org tree service.

See Also:
Constant Field Values

ATTR_GROUP_TYPE

static final java.lang.String ATTR_GROUP_TYPE
Attribute used to flag if this OrgNode group is a non LDAP OrgNode group for display purposes. For example, dummy container nodes can be used in the attribute based org tree to contain the root person nodes

See Also:

ATTR_RETRIEVED_CHILDREN

Attribute used by the cache to indicated if the children of this node have been retrieved. Not applicable outside the context of the cache.
