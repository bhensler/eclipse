com.lotus.sametime.genericadaptermanager.cluster

Class ServerManager

  • java.lang.Object
    • com.lotus.sametime.genericadaptermanager.cluster.ServerManager
  • All Implemented Interfaces:
    LoginListener, TimerListener, java.util.EventListener

    public class ServerManager
extends java.lang.Object
implements LoginListener, TimerListener
    This class represents a login to a ST server. The generic adapter logins as Server Application to each configured ST server. Upon initialization, this class will perform the login. Each login is represented by a ST session. If the ServerManager didn't manage to login OR logged out not per our request, a timer will be triggered in order to try to re-login.
