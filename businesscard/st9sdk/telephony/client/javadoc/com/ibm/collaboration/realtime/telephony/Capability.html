Interface Capability

All Known Subinterfaces:
Call, CallManager, CallServiceProvider, CapabilityManager, ClickToCallServiceProvider, Participant, VideoChatServiceProvider, VoiceChatServiceProvider
All Known Implementing Classes:
AbstractCallServiceProvider, AbstractClickToCallServiceProvider, AbstractVideoChatServiceProvider, AbstractVoiceChatServiceProvider
public interface Capability

Capability defines capabilities that can be supported by a telephony service provider for an individual call or participant. This interface is extended by other interfaces that support the ability to query capabilities.
