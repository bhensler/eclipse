com.lotus.sametime.cachedresolvemanager.resolve

Class FileMemorySyncMgr

  • java.lang.Object
    • com.lotus.sametime.cachedresolvemanager.resolve.FileMemorySyncMgr
  • All Implemented Interfaces:
    TimerListener

    public class FileMemorySyncMgr
extends java.lang.Object
implements TimerListener
    This component performs periodical cleaning the memory cache and synchronization the memory cache with the resolve data stored in the persistent file. It has two main activities: Load the persistent file to memory and flush the memory back to the file.

      • flushMemoryToFile

        public void flushMemoryToFile()
        Writes the memory cache content to the persistent file. Performs the operations: 1. Looks for the cached file folder; creates it if doesn't exist. 2. If the folder contains more than _filesCount files, deletes the oldest ones. 3. Creates a new version of the persistent file whose name contains the current time stamp. 4. Flush the cache content into the newly created file.
