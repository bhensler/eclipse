    public class ChatUIComp
extends STComp
implements ChatUI, InviteDialogListener, InvitationListener, JoinMeetingListener, LoginListener
    The interface defines the Instant Message (IM) manager UI component. This component manages the creation of 1-on-1 chats (IM), chat confrences and meetings.
