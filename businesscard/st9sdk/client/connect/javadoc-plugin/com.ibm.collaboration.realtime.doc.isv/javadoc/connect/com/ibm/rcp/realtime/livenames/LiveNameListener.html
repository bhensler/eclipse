statusChanged

void statusChanged(java.lang.String lookupName,
                   LiveName livename)
Called when a lookup name is resolved into a LiveName object and thereafter to establish and update the status of a livename. Implementators can use this livename object to display awareness.

Parameters:
lookupName - the lookupName from which the livename object was resolved.
livename - the livename object whose status changed

resolveFailed

void resolveFailed(java.lang.String lookupName,
                   java.lang.String communityId)
Called when a lookup name fails to be resolved to a LiveName object for the indicated community. If offline communities remain and are logged into, they will be searched and the name may yet be resolved into a LiveName object.

Parameters:
lookupName - the lookup name for which the resolve failed.
communityId - the community ID for which the resolve failed.
