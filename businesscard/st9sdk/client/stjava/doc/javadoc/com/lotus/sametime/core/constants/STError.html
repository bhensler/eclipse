com.lotus.sametime.core.constants

Class STError

  • java.lang.Object
    • com.lotus.sametime.core.constants.STError

  • public class STError
extends java.lang.Object
    Sametime version 2.0 error codes.
    High bit is 1 to indicate an error.
    Low 2 bytes are the generic error code (understood by everyone).
    High 2 bytes - service specific error code (provided in addition to the generic error code for those who can interpret it).
