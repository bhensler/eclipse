com.lotus.sametime.guiutils.chat

Class ChatParagraph

  • java.lang.Object
    • com.lotus.sametime.guiutils.chat.ChatParagraph
  • Deprecated. 
    due to the availability of additional non-AWT frameworks. This AWT component is no longer supported.

    public class ChatParagraph
extends java.lang.Object
    This is the object representing a paragraph of text in the chat transcript. The paragraph compiles its line in the m_lines Vector containing objects of type ChatLine.
