    public final class ChatTranscript
extends java.awt.Panel
implements java.awt.event.AdjustmentListener, java.awt.event.KeyListener, java.awt.event.ActionListener
    The chat transcript panel, embedded in the ChatArea ScrollPane. It consists of two columns, the name column (name of user sending the text), and the transcript column (the text itself)
