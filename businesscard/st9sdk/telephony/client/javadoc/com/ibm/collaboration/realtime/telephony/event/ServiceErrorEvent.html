Interface ServiceErrorEvent

All Superinterfaces:
CallEvent, java.io.Serializable, TelephonyEvent
public interface ServiceErrorEvent
extends CallEvent

ServiceErrorEvent is a call event used to signal an error from the telephony service provider. See CallEvent for more information.
