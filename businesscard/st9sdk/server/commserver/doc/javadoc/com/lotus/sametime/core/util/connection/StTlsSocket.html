Class StTlsSocket

  • java.lang.Object
    • java.net.Socket
      • com.lotus.sametime.core.util.connection.StTlsSocket
  • All Implemented Interfaces:
    java.io.Closeable, java.lang.AutoCloseable

    public class StTlsSocket
extends java.net.Socket
    This is a TLS (Transport Layer Security) socket, much like the standard SSLSocket class, only this class is implemented by wrapping an StTlsEngine.

           java.net.InetAddress localAddress,
           int localPort)
            throws java.io.IOException
        Constructor for a client socket that connects to the specified remote address and port. This does not include TLS handshake. The socket is also bound to the local address and port, if supplied.
        Parameters:
        config - Configuration settings.
        remoteAddress - The server address to connect to.
        remotePort - The server port number to connect to.
        localAddress - The local address to bind to. Pass null to bind to any local address.
        localPort - The local port to bind to. Pass 0 to bind to any local free port.
        Throws:
        java.io.IOException - If an I/O error occurs at the TCP/IP layer while attempting to bind to the local address, or while attempting to connect to the remote address.

      • StTlsSocket

        public StTlsSocket(StTlsConfig config,
           java.net.Socket tcpSocket,
           boolean autoClose)
        Constructor for a TLS socket that wraps an existing TCP/IP socket. When closing this TLS socket, the wrapped TCP/IP socket will get closed automatically.
        Parameters:
        config - Configuration settings.
        tcpSocket - The underlying TCP/IP socket. If null, this constructor creates an unconnected client socket.
        autoClose - Close tcpSocket when this socket gets closed.
