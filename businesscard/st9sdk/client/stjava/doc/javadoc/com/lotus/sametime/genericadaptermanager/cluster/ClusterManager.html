Class ClusterManager

  • java.lang.Object
    • com.lotus.sametime.genericadaptermanager.cluster.ClusterManager
  • All Implemented Interfaces:
    ServerManagerListener

    public class ClusterManager
extends java.lang.Object
implements ServerManagerListener
    This class manages all the logins to ST servers in a non/clustered ST environment.
    It is responsible to get the ST servers details from ST Configuration and connect to each one of them. Each login is represented by a ServerManager object In order to be informed on community events (connected/disconnected), the application should implement the CommunityListener interface. In order to be informed on server events (logged in/out), the application should implement ServerLoginListener.
