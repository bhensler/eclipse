public class MyMCU
extends java.lang.Object
implements MyAVConstants

Simple "fake MCU" that coordinates the handling of requests from and notifications to MyAVConferenceService. This communication between the conference service and MCU passes by way of the MyMCUCommandSocketServer running on the MCU server (siplet) and the MyMCUCommandSocketClient running on the Sametime Media Server.
