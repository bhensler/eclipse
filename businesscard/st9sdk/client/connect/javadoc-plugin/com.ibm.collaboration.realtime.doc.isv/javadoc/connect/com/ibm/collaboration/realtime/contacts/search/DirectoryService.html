queryUserAttributes

void queryUserAttributes(java.lang.String contactId,
                         java.lang.String[] attributes)
Requests directory information for the specified user. The requested information consists of one or more directory attributes as defined in the DirectoryInfo interface. Before calling this method, use addUserInfoLookupListener(UserLookupListener) to register a user info lookup event listener. The requested directory information will be sent to that listener in a UserInfoLookupEvent object; use UserInfoLookupEvent.getUserInfo() to access the requested information.

Parameters:
contactId - contact ID of user to query
attributes - array of attribute names as defined in DirectoryInfo
