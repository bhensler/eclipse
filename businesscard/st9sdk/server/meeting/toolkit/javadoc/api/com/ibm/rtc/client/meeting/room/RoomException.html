Serializable
public class RoomException
extends BaseException

Used to distinguish exceptions related to the use of the room management REST API, where the remote client is able to communicate with the server, but the server is not able to fulfill the request for some reason. The methods on this class provide the details needed to distinguish between several different types of issues.
