Interface ChatFactory

  • All Known Implementing Classes:
    DefaultChatFactory
    Deprecated. 
    due to the availability of additional non-AWT GUI frameworks. This component is no longer supported.

    public interface ChatFactory
    Interface for the factory used for creating views. The ChatUI component uses the factory for generating the required UI windows and alerts. Separating the creation of the views from the ChatUI component enables the creation of customized views by supplying different implementation of the factory.

      • createView

        void createView(ChatModel model,
              java.lang.String partnerName,
              boolean originator)
        Deprecated. 
        Creates the chat window for the given model. Called by the ChatUI component when a new view is required for display of an new chat session. The request can be trigged by the user initating a new chat session or a session initiated by the other user/s in the community.
        Parameters:
        model - The model that will be used by the view.
        partnerName - The partner name, if this chat model represents an IM. This is only relevant for 1 on 1 chat.
        originator - True if this client is the initiator of the chat. This is only relevant for 1 on 1 chat.
