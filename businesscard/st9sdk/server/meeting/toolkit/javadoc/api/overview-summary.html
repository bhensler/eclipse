This documentation describes the IBM® Sametime® Meeting Remote Client APIs

          Description

Packages
com.ibm.rtc.client.http Provides generic classes necessary to Define the most common and basic HTTP services needed by REST APIs. Abstract the caller from the lower-level toolkit being used. Note, these classes are not tied to any single product, rather these are intended to work as a re-usable base for any REST API.
com.ibm.rtc.client.meeting.annotations Classes that deal with pointers and highlighting on the projector.
com.ibm.rtc.client.meeting.appshare Classes that deal with screen sharing.
com.ibm.rtc.client.meeting.chat Classes that deal with messages and tags in the discussion window.
com.ibm.rtc.client.meeting.configuration Dealing with server configuration data.
com.ibm.rtc.client.meeting.docshare Provides the classes representing the Document component of meetings.
com.ibm.rtc.client.meeting.participants Classes related to the participants list and user status in the room.
com.ibm.rtc.client.meeting.policy Dealing with user policy data as set by the server.
com.ibm.rtc.client.meeting.polling Classes related to creating and handling polls, and responding to polls.
com.ibm.rtc.client.meeting.projector Classes related to the projector component of the meeting, giving access to information about the projector and event handling.
com.ibm.rtc.client.meeting.room Provides the classes necessary to create and manage meeting rooms on the Sametime Meeting Server.
com.ibm.rtc.client.meeting.summary The summary services allow the caller to gather data about events that have occurred in a room.
com.ibm.rtc.client.meeting.url Classes related to storing URLs, and working with URLs.
com.ibm.rtc.client.meeting.userinfo The user info services allow the caller to gather data about known users of the system, this is not a way of searching for users.
com.ibm.rtc.client.rtc4web Classes that handle realtime operations controlling session state.
com.ibm.rtc.client.utils Contains some utility methods for working with JSON.
