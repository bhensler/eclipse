Interface PropertyNames

public interface PropertyNames

PropertyNames defines the names of telephony API properties that are passed to or from the service provider. The PROPERTY_TCSPI_xxx properties correspond to Sametime Telephony Conferencing SPI configuration options.

PROPERTY_TCSPI_EXPECTED_USERS

static final java.lang.String PROPERTY_TCSPI_EXPECTED_USERS
Expected number of users in a conference. This is passed to the TCSPI service provider so that the provider to select an appropriate telephony service to host the call. Sametime Connect sets this value to the number of users initially invited to the call (including the initiator). However, participants can invite additional users to join the call unless the service provider prevents it by setting the maximum user count to the expected number of users.

Since:
Sametime 8.0

PROPERTY_TCSPI_VIDEO_IN_USE

static final java.lang.String PROPERTY_TCSPI_VIDEO_IN_USE
TCSPI property indicating whether external video is in use for a given call. When the client detects this call property with the value true, it will adjust the call window title to reflect that the call is a video call. This optional property allows TCSPI implementations, if they wish to do so, to indicate the use of external video that the client would not be aware of otherwise. This property can be published at any point after the call begins.

Since:
Sametime 8.5.1
See Also:
