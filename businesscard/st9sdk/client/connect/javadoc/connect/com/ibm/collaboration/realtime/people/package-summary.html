Package com.ibm.collaboration.realtime.people

Interface Summary
Group Group represents a group in a contact list.
IGroupService GroupService provides methods to manage groups.
NestableGroup NestableGroup represents a contact list group that can contain other groups, for example, a private group.
PeopleService PeopleService provides methods to manage people.
Person Person represents a person in a contact list.
PersonListener PersonListener is used to process Person lifecycle events
PrivateGroup PrivateGroup represents a private group in the contact list.
PublicGroup PublicGroup represents a public group in the contact list.
 

Class Summary
PersonGroupTransfer PersonGroupTransfer provides a mechanism for converting an array of Sametime Buddylist Person / Group objects to a platform specific representation of the data and vice versa.
