Class InvitationManager

  • java.lang.Object
    • com.lotus.sametime.chatui.invitation.InvitationManager
  • All Implemented Interfaces:

    Deprecated. 
    due to the availability of additional non-AWT GUI frameworks. This component is no longer supported.

    public class InvitationManager
extends java.lang.Object
implements PostServiceListener, ImServiceListener, LoginListener, STEventListener
    This class manages incoming and outgoing invitations regarding meetings. Currently (watson release), we support both the deprecated conference semantic protocol as well as the post internal protocol. This object makes sure that the ChatUI component doesn't have to trouble it's pretty little head with that mess.
