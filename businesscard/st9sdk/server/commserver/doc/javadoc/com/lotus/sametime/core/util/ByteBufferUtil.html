      • prepareWrite

        public static void prepareWrite(java.nio.ByteBuffer buffer)
        Call this when switching the buffer from read to write mode. It is similar to Buffer.clear(), except it does not discard unread bytes. By definition, it discards all the bytes that were already consumed (up until the current position) by shifting the unread bytes all the way to the left, then it sets the position one byte beyond the last unread byte, and the limit is set to the capacity.
        Parameters:
        buffer - The buffer to switch from read to write mode.

      • unflip

        public static void unflip(java.nio.ByteBuffer buffer)
        The opposite of Buffer.flip(). While flip() switches from write to read mode, this call switches back from read to write mode. Specifically, it sets the buffer's position to the current limit, and the buffer's limit to the current capacity. This is useful when reading data and determining that more data is needed: 
         1. Write to a buffer.
 2. Flip the buffer.
 3. Read from the buffer, and realize that more data is needed.
 4. Unflip the buffer.
 5. Goto 1.
        Parameters:
        buffer - The buffer to unflip.

      • ensureCapacity

        public static java.nio.ByteBuffer ensureCapacity(java.nio.ByteBuffer buffer,
                                 int size)
        Call this method before writing to a buffer, to ensure the buffer has enough bytes remaining. This call returns the same buffer if it has enough room, otherwise replaces it with a larger buffer. Either way, the caller should continue working with the returned buffer.
        Parameters:
        buffer - The buffer to write to. If there are enough bytes remaining in this buffer, then it is not modified. Otherwise, upon return, its position is not changed, but it has 0 remaining bytes. A null reference is allowed.
        size - Minimum number of bytes to have remaining.
        Returns:
        The newly allocated buffer, or the same instance as the buffer argument if there was no need to allocate a new one.
