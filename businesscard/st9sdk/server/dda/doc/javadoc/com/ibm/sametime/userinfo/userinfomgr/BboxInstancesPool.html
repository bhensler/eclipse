Class BboxInstancesPool

  • java.lang.Object
    • com.ibm.sametime.userinfo.userinfomgr.BboxInstancesPool

  • public class BboxInstancesPool
extends java.lang.Object
    This class represents a pool of black box instances. The Instances are bb objects that represent a connection to a db/ directory and thie number is limited the Instances will be created , initialized and stored in a data structure. when tha application needs to perform an operation that requires accessing a db or directory it will ask an instance from the pool and return it when the operation completes.
