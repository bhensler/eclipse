      • createIm

        Im createIm(STUser stuser,
          EncLevel encLevel,
          int imType,
          boolean preferLoginId)
        Creates an IM session with a specified user.
        Parameters:
        stuser - The user to whom the IM session should be opened.
        encLevel - The level of encryption of the IM. ENC_LEVEL_ALL (Any encryption); ENC_LEVEL_DONT_CARE (Doesn't matter if message is encrypted); ENC_LEVEL_NONE (No encryption); ENC_LEVEL_RC2_40 (Encryption using RC2 required.)
        imType - The IM type. Types 1-100,000 are reserved.
        preferLoginId - If true then preferred option for creating the IM will be first to the login id. If the login id is not available (the passed STUser reference is not pointing to a STUserInstance object) the IM will be created to the user id. If false the Im will always be created to the user id and not to a specific login.
        Returns:
        The IM object used for sending and receiving instant messages.
