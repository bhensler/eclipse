executeCommand

java.lang.Object executeCommand(java.lang.String commandId,
                                java.lang.Object options)
                                throws TelephonyServiceException
Executes a provider-specific command for this participant, if supported by the service provider. This mechanism is used to provide third-party plug-ins with the ability to access services that are not specifically supported by Sametime Connect. The command ID and the data types for the input options and return value are determined by the service provider.

Parameters:
commandId - command ID
options - command options
Returns:
Command result, null if no return value is required.
Throws:
InvalidParameterException - if a required parameter is null or invalid
UnsupportedCapabilityException - if this operation is not supported
TelephonyServiceException - if the operation fails for some other reason
Since:
Sametime 7.5.1
See Also:
Call.executeCommand(String, Object), CallManager.executeCommand(String, Object)
