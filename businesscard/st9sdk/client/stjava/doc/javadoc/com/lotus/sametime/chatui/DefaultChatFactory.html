      • getCustomizedPanels

        public java.awt.Panel getCustomizedPanels(int panelPosition,
                                 ChatFrame frame)
        Deprecated. 
        Called by the ChatFrame to get customized user panels. The default implementation does not provide additional panels. Customized versions of this factory should overide this method and supply panels to be placed within the default chat frame. The ChatFrame will call on this method 3 times, each time specifiying a different panel position according to the constants defined.
        Parameters:
        panelPosition - One of the fixed positions defined by DefaultChatFactory.TOP_PANEL,DefaultChatFactory.CENTER_PANEL, DefaultChatFactory.BOTTOM_PANEL
        frame - The chat frame that the panel will be embedded in.
        See Also:
        TOP_PANEL, CENTER_PANEL, BOTTOM_PANEL

      • getTextModifier

        public TextModifier getTextModifier(ChatFrame frame)
        Deprecated. 
        Called by the ChatFrame to get the TextModifier that will be associated with the given ChatFrame. The default implementation does not provide a TextModifier (returns null). The customized version of this factory should overide this method and supply the custom TextModifier. TextModifiers are used to intercept text messages and optionaly modify them before dispatching the message to the chat session (e.g. spell checking).
        Parameters:
        frame - The ChatFrame that the TextModifier will be associated with.
        Returns:
        TextModifier The TextModifier that will be used for modifying messages or null if not available.
