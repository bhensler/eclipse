setProperties

void setProperties(java.util.Properties properties)
Sets the call properties required by the service provider. The properties required by IBM telephony service provider plug-ins are defined are defined in PropertyNames. In addition, custom properties can be passed to a TCSPI call service provider implementation when starting a call: Any properties that are not defined in PropertyNames are assumed to be custom properties, and are set as call properties as soon as the call starts. Note that the client implementation may modify the call properties while establishing the call.

Parameters:
properties - properties required by service provider

setMaximumVideoBitRate

void setMaximumVideoBitRate(int kbps)
Sets the maximum video bit rate in kilobits per second (Kbps). The maximum bit rate is defined by a user policy on the Sametime server, and should be passed to the service provider for each call via this method. The default value of 0 means no maximum video bit rate policy has been specified.

Parameters:
kbps - maximum bit rate in Kbps

void setVideoCall(boolean flag)
Specifies whether this is a video call.
