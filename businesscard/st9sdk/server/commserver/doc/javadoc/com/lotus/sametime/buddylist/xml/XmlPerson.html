      • belongsToDefaultCommunity

        boolean belongsToDefaultCommunity()
        Returns whether or not this person belongs to the default community.

        In Sametime 7.5, the Sametime client started to support multiple communities. The XmlBuddyList was introduced to support having a buddylist with users and groups from multiple communities within it. When you use the Java Toolkit and the XmlBuddyListService directly, you are working within a single community and the XmlBuddyList you retrieve is from that default community. This method indicates whether this person belongs to that default community you are logged into.

        This is useful when resolving and displaying users because you only want to attempt to resolve those users for the community you are logged into, and this method helps you filter those out.
        Returns:
        Returns true if this person belongs to the default community, false otherwise.
        Since:
        Sametime 8.5
