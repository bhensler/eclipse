      • subscribe

        public int subscribe(STId groupId)
        Subscribe for a group We send subscribe for a group only for the first subscriber. all listeners subscribed by then will receive group content. For the next subscribers we do not send an event and they will not receive group content but will be notified on changes in the group
        Specified by:
        subscribe in interface GroupSaService
        Parameters:
        groupId - - the STId of the group
        Returns:
        - reason code : 0 means subscribe event sent to the server and the listeners already subscribed will receive groupcontent 1 means subscribe request for this group has already been sent to the server new subscriber will not receive group content for this group but will receive events for changes in the group.
