Interface CapabilityManager

All Superinterfaces:
CallListener, Capability
public interface CapabilityManager
extends Capability, CallListener

CapabilityManager manages capabilities associated with a specific call. This extends the existing Capability.isCapabilitySupported(String) mechanism by allowing UI code to listen for capability changes and update accordingly, and by allowing service providers to indicate when a capability has changed. See also CapabilityEvent.

Since:
Sametime 8.5.2

addPropertyChangeListener

void addPropertyChangeListener(java.beans.PropertyChangeListener listener)
Adds a property change listener for all capabilities in the call associated with this capability manager. The listener will receive a PropertyChangedEvent with the following values:

property name: capability name as defined in the Capability interface

new value: Boolean, true if capability is supported, false if not supported

old value: null

Note that receiving a property change does not guarantee that the capability value has actually changed. Note also that capability changes can be tracked using CapabilityEvent.

Parameters:
listener - property change listener to add

broadcastSupportedCapabilities

void broadcastSupportedCapabilities()
