com.ibm.sametime.userinfo.sa

Class STUserInfoSA

  • java.lang.Object
    • com.ibm.sametime.stjavautils.sa.STBaseServerApplication
      • com.ibm.sametime.userinfo.sa.STUserInfoSA

  • public final class STUserInfoSA
extends com.ibm.sametime.stjavautils.sa.STBaseServerApplication
    UserInfo Server application is an additional mean (apart from the servlet) to obtain Users' data from the corporate directory. This class performs as an additional server application. It logins to the community server and accepts clients request via vp protocol. Then , as the servlet, it forwards to the request to the UserInfoMgr singleton which in turn forward it to the loaded blackboxes. The response received from the blackboxes is received at this object as an xml string and is being sent back to the client.
