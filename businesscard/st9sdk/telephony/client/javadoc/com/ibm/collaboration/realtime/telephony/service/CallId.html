setIsConferenceCall

public void setIsConferenceCall(boolean flag)
Deprecated. 
Specifies whether the call is a conference call. Note that the CallManager implementation will automatically set this field to true for calls that are started with more than two parties, or when connecting to an incoming call identified as a conference call. For two party calls that have conference moderator controls, the service provider should set this to true in the CallId object returned from startCall. The CallManager implementation will never set this value to false.

Parameters:
flag - true if call is a conference call, false otherwise
Since:
Sametime Unified Telephony 8.0, Sametime 8.5
