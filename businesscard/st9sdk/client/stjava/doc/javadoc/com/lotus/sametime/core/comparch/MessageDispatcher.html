com.lotus.sametime.core.comparch

Class MessageDispatcher

  • java.lang.Object
    • com.lotus.sametime.core.util.Queue
      • com.lotus.sametime.core.comparch.MessageDispatcher

  • public class MessageDispatcher
extends Queue
    The Sametime event queue. The name of this class is a bit misleading - This class no longer does the dispatching, as it did in 2.0. It's just the event queue. But the name couldn't change because of compatibility. The dispatching is done by The MessageDispatchingThread class.

        public void flush()
