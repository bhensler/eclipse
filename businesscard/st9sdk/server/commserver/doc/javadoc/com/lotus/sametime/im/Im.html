com.lotus.sametime.im

Class Im

  • java.lang.Object
    • com.lotus.sametime.im.Im

  • public class Im
extends java.lang.Object
    Represents a single Instant Message (IM) session to a specified user. Enables sending and receiving a message to/from the user on the other side of the session.
