Auction House
Description:

The Auction House sample implements an auction house using the Sametime Java and Community Server toolkits. The purpose of this sample is to show how, by using the Server Toolkit, one can simplify applications and make them more robust and flexible. In this sample, the auctioneer is implemented as a server side activity in the place.

See the IBM Redbook, Working with the Sametime Community Server Toolkit, for information on using the sample.

API Used:

PlacesAdminService, ServerAppService
