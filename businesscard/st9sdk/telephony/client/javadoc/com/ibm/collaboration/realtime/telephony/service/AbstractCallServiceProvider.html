Deprecated. For internal use only starting with Sametime 8.5.2. 

@Deprecated
public abstract class AbstractCallServiceProvider
extends java.lang.Object
implements CallServiceProvider

AbstractCallServiceProvider provides default implementations for methods in the CallServiceProvider interface. Default implementations, which throw an UnsupportedCapabilityException, are provided for methods that represent optional features, or new features added since the original implementation of this class for Sametime 7.5. Service providers who implement CallServiceProvider should extend AbstractCallServiceProvider and provide implementations for the appropriate methods. Use Capability.isCapabilitySupported(String) to declare the capabilities that correspond to the implemented methods.

Since:
Sametime 7.5.1

                   throws TelephonyServiceException
Deprecated. 
Description copied from interface: CallServiceProvider
Sends a DTMF (dual-tone multifrequency) event representing a key on a telephone keypad, if supported by the service provider. The event is processed in the context of an existing call, and is represented by an RFC 2833 DTMF event code, one of the following: 0-9 for the corresponding digits, 10 for '*', 11 for '#', and 16 for the the Flash key. This method can be used by service providers who need to process telephone key presses as they occur. To process a string of telephone keypad digits all at once, use CallServiceProvider.sendDtmfDigits(CallId, String) instead.
