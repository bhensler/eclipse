Class StTlsSocketFactory

  • java.lang.Object
    • javax.net.SocketFactory
      • javax.net.ssl.SSLSocketFactory
        • com.lotus.sametime.core.util.connection.StTlsSocketFactory

  • public class StTlsSocketFactory
extends javax.net.ssl.SSLSocketFactory
    This factory creates TLS client sockets of type StTlsSocket. Note that StTlsSocket has a public constructor, allowing to create sockets without this factory just as well.

    To create a new StTlsSocketFactory you will first need to create an StTlsConfig object.
