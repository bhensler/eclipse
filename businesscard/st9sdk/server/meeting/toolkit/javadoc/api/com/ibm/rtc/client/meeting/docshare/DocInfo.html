fetchPage

public DocPage fetchPage(RealTimeSession rtSession,
                         String sid,
                         String docId,
                         long pageNum,
                         boolean retainBitmap)
                  throws IOException,
                         DocShareException
Fetches the specified page # for the document. Image bytes may/may not be retained with a page fetch. This can be customized via the retainBitmap flag. Note: Applications may run out of memory if too many images are stored in memory.

Parameters:
rtSession - RealTimeSession to which this document belongs
sid - Session Id
docId - Document Id
pageNum - Page to be fetched
retainBitmap - True if image bytes are to be retained. False, if not.
Returns:
DocPage Object representing a document page.
Throws:
IOException
DocShareException - Thrown if an unexpected response is encountered from the server.
