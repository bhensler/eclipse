        public void enableAutomaticReconnect(int reconnectAttempts,
                            long timeout)
        Enable automatic reconnect to server. When enabled the client will attempt to reconnect to the server continuously till one of the conditions is reached: 1. A successful login to the server. 2. The number of specified attempts is completed with no success. The automatic reconnect will function only in the case where the client is disconnected unexpectedly from the server and not in the case where the client explicitly requires to logout.
