    public class StTable
extends java.awt.Container
implements java.awt.ItemSelectable
    Table is a user-interface component that presents data in a two-dimensional table format.
    This Table supports dynamic table data change through the TableModel interface provided in the constructor. Whenever any data got changed, call 
     resizeAndRepaint()
    to refresh the view.
    When row selection events occur, Table fires ActionEvent (for mouse d-clicks) and ItemEvent when row is selected. Selection sensitivity can be set through 
     setSelectionAllowed(boolean cond)
    wrapper-method.
    Table allows changing autoresize mode with 
     setAutoresizeMode(int mode)
    . Two modes are supported in this implementation: 
     AUTO_RESIZE_LAST_COLUMN
    and 
     AUTO_RESIZE_NEXT_COLUMN
    .
    Table grid presence and rendering can be controlled by the following methods:
     setShowVerticalLines(boolean cond)
    - to controls presence of vertical lines in a grid; 
     setShowHorizontalLines(boolean cond)
    - to controls presence of horizontal lines in a grid; 
     setGridColor(Color c)
    - to set grid color;
    Other rendering options can be defined by: 
     setBackground(Color c)
    - to set Table background color; 
     setForeground(Color c)
    - to set Table foreground color; 
     setFont(Font f)
    - to set Table display font; 
     setSelectionBackground(Color c)
    - to set selection background color; 
     setSelectionForeground(Color c)
    - to set selection foreground color.
