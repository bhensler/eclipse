Interface CommunityListener

public interface CommunityListener

Listener for community-related events.

Note:

A single thread is responsible for invoking all community listeners, so it's very important that implementors of this interface do not block. It is recommended that implementations schedule Jobs in response to community events.

Since:
Sametime 7.5.1
