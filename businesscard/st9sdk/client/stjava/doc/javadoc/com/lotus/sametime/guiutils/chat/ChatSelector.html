Class ChatSelector

    public class ChatSelector
extends java.lang.Object
implements java.awt.event.MouseListener, java.awt.event.MouseMotionListener, java.awt.event.FocusListener
    This is the object managing the text selection procedures in its host ChatTranscript. Comments in this class use the term "session". A session starts when the user clicks the mouse inside the transcript and continues until the mouse is released. Between the click and the release the user might have dragged the mouse.
