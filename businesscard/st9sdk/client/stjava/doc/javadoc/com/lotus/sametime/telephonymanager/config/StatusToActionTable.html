com.lotus.sametime.telephonymanager.config

Class StatusToActionTable

  • java.lang.Object
    • com.lotus.sametime.telephonymanager.config.StatusToActionTable

  • public class StatusToActionTable
extends java.lang.Object
    This is a singleton that manages the required action for each Telephony attribute. It gets the proper action form the configuration file. The current default behavior is: Telephony Status Operation ---------------- ------ TELEPHONE_STATUS_UNKNOWN (0) not supported (-1) TELEPHONE_STATUS_AVAILABLE (1) remove attribute (1) TELEPHONE_STATUS_BUSY (2) set attribute (0) TELEPHONE_STATUS_DO_NOT_DISTURB (3) not supported (-1)
