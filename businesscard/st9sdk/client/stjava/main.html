Welcome to the Sametime 9.0 Java Toolkit

The Sametime Java Toolkit provides a unified API for the Sametime Community and Meeting services, making it easy for both novice and experienced Java developers to embed real-time collaboration technology into new or existing applets, applications, and web sites.

The Sametime Java toolkit provides APIs to all the client features available in Sametime and can be used with any standard Java development environment supporting JDK 1.4.2 or higher.
