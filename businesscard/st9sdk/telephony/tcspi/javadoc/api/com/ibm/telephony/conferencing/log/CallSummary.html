com.ibm.telephony.conferencing.log
Class CallSummary

java.lang.Object
  com.ibm.telephony.conferencing.log.CallSummary

public class CallSummary
extends java.lang.Object

CallSummary is a class that allows us to log to a separate log file call summary information after a telephony call as ended To create the CallSummary log, use Logger.getLogger(CallSummary.LOGGER_NAME); The Conference log file should be in the regular log directory with the name conference.log. If it it can't find the log directory, it will be put in the temp directory.
