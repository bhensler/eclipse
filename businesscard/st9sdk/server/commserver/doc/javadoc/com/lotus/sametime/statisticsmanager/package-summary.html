    StatisticsManager
    This class is used to log performance counters. 1) It is located in the toolkit (and not in stjavautils package) as the toolkit use it for its own counter and it should be depended in other jars. 2) Each SA e.g., offline messages SA need to define its counters in a class e.g., OfflineMessagesCounters.java. 3) The table of counters should be passed to the manager init method which also start the logging. 4) we used Java Util Logger for the counters log in order to have logs that recycle 5) In order to add a new counter - add it to the SA counters class, and call it from getCounterXXXX method where you need access to the counter 6) the manager uses a timer thread to query all the counters and logs their values
