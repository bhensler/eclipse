public class MeetingRoom
extends HashMap<String,Object>

This class represents a meeting room with all its attributes. A room name or a permaName (permanent identifier for the room usually derived from the room name) is required for room creation. All other attributes are optional and can be added/edited later. The room name is also editable. However, the permaName is immutable.
