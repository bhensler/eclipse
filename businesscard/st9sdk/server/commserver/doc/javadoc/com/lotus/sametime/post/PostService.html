    public interface PostService
extends STCompApi
    Defines the post service used for sending one time messages to other users and optionally receiving responses from those users.

      • createPost

        Post createPost(int postType,
              java.lang.String title,
              java.lang.String message,
              int detailsType,
              byte[] details,
              STUser[] users,
              boolean preferLoginId)
        Creates a post object with the specified post type, title, message, details type, details and a list of users to be posted by the object.
        Parameters:
        postType - The type of the message post.
        title - The title of the post.
        message - The post message.
        detailsType - The type of the additional details.
        details - Additional post information.
        users - The list of "postees".
        preferLoginId - If true then preferred option for creating the Post will be first to the login id. If the login id is not available (the passed STUser reference is not pointing to a STUserInstance object) the Post will be created to the user id. If false the Post will always be created to the user id and not to a specific login.
        Returns:
        A Post object that will be used to perform the message post.
