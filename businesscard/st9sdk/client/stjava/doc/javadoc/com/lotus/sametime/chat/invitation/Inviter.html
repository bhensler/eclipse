        Invite the given users to a meeting.
        Parameters:
        meetingInfo - The meeting details.
        inviteText - The invitation message.
        users - The users to invite.
        preferLoginId - If true then the preferred option for creating the invite will be first to the login id. If the login id is not available (The passed STUser reference is not pointing to a STUserInstance object) the invitation will be created to the user id. If false the invitation will always be created to the user id and not to a specific login.
