Interface OrgTreeCacheProvider

All Superinterfaces:
OrgTreeProvider, OrgTreeWalker
public interface OrgTreeCacheProvider
extends OrgTreeProvider

Interface which provides the caching functionality used by the organization tree SA and also by the client. In otherwords, the orgTreeCacheProvider can be used to manage the cache for the SA and the client. This interface is for internal use only.

getRootNode

OrgNode getRootNode(java.lang.String requestorContactId,
                    boolean useShallowCopy)
                    throws javax.naming.NamingException
Return the cached root node. The organization tree SA uses a shallow copy so as not to pass the fully cached tree over the wire to remote callers. The client on the other hand uses full copies in order to restore the fully cached tree in the organization view UI.

Parameters:
requestorContactId - the contact ID of the person making this request.
useShallowCopy - Whether or not to return OrgNodes that have no children (a shallow copy of the OrgNode).
Returns:
