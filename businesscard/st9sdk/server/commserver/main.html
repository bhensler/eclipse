Welcome to the Sametime 9.0 Community Server Toolkit

Sametime includes a comprehensive, Java-based application development toolkit that developers use to add and manage real-time capabilities in their e-business applications. The Sametime Toolkits are a collection of building blocks or components used by developers to build applications that affect the functionality and services provided by a Sametime Server. These Toolkit components can be used in any JDK 1.7 Java development environment. The code should be compiled with JDK 1.6 to allow running on Domino server, since Domino runs with 1.6 JRE.
