      • setSynchronizeBuddyLists

        public void setSynchronizeBuddyLists(boolean synchronize)
        Description copied from interface: XmlBuddyListService
        Specify whether the XmlBuddyListService should synchronize all buddylist activities between the new XML format and the old classic format. If synchronization is enabled, you can expect the following behavior: - XmlBuddyLists when retrieved from storage using the @see XmlBuddyListService.getBuddyList() method will be compared to the classic version and synchronized if necessary. - Changes to XML will be stored to the classic format in storage when @see XmlBuddyListService.saveBuddyList(XmlBuddyList) is called.
        Specified by:
        setSynchronizeBuddyLists in interface XmlBuddyListService
        Parameters:
        synchronize - true to enable synchronization of buddylists, false otherwise. The default is true.
