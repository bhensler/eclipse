com.ibm.sametime.userinfo.userInfobbapi

Interface UserInfoBlackBoxAPI

  • All Known Implementing Classes:
    UserInfoLdapBB, UserInfoNotesBB, UserInfoNotesCustomBB

    public interface UserInfoBlackBoxAPI
    Defines the API for providing user information in a synchronous way. It consists of an initialization and termination stages represented by the init() and terminate() methods respectively. These methods should only be run once. init() should be run when the class loads in order to perform initialization steps. The terminate() method should be run just before the object is destroyed, in order to close the connections and perform additional cleanup operations. The main activity is performed using the processRequest() method which is used to query the storage and handle the response received.
