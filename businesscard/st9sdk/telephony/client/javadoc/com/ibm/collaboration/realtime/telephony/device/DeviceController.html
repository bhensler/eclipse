setFeature

void setFeature(java.lang.String featureId,
                java.lang.Object settings)
                throws TelephonyServiceException
Deprecated. 
Changes the settings of the specified device feature, if supported by the service provider. This mechanism is used to provide third-party plug-ins with access to device features that are not specifically supported by Sametime Connect. Both the feature ID and the feature settings data type are determined by the service provider. For features that have only two states (enabled and disabled) and no other settings, use setFeatureEnabled(String, boolean) instead.

Parameters:
featureId - feature ID
settings - feature settings
Throws:
InvalidParameterException - if either parameter is null or invalid
UnsupportedCapabilityException - if this operation is not supported
TelephonyServiceException - if the operation fails for some other reason
