      • setProtocolVersion

        public void setProtocolVersion(int newVersion)
        Set The protocol version of the service this channel is connected to. This is only meaningful if called before accepting the channel, to reduce the protocol version. In all other cases, this call will be ignored. That is, the receiving party can choose to accept the channel, but state that it only supports an older protocol version.
        Parameters:
        newVersion - The protocol version of the service this channel is connected to.
