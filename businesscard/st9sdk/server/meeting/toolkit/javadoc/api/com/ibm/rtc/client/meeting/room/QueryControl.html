java.lang.Object

public class QueryControl
extends Object

This class provides parameters to control the scope of search queries, the size and ordering of the resultant set. To get the default search parameters, the static getDefault() method can be used. 

 QueryResults results = roomServices.getCallersRooms(QueryControl.getDefault());

getDefault

public static QueryControl getDefault()
Returns an immutable QueryControl object with default search parameters set. The defaults are:
  • Rooms are sorted based on room name
  • Sort order is ascending.
  • Hidden rooms are not included in the search.
  • A set of 10 matching results will be returned starting with index 0
Note: This is an immutable instance. Calling the setters on this instance will result in an IllegalAccessException. A new QueryControl instance needs to be used to customize the parameters. This class is not thread safe.

QueryControl The default immutable instance
