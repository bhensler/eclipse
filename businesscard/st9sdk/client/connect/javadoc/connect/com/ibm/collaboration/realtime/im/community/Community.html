com.ibm.collaboration.realtime.im.community
Interface Community

public interface Community

Community represents a instant messaging and presence community. A community is defined both the server host name and the protocol. For example, if you have separate (non-clustered) Sametime servers in your organization, each one is considered a different community.

getServerCommunityId

java.lang.String getServerCommunityId()
Returns the ID of the messaging server that this community represents. This ID, if present, is the same for all servers in the same community, unlike the host name. The ID was introduced in ST8.5 and is only available from 8.5 servers. By default the server ID returned is null.

Returns:
an ID of the community this server represents, or null
