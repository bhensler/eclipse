public static Policy getPolicyService(ServiceBase serviceBase,
                                      String id,
                                      Policy.Type type)
                               throws IOException,

A static method, to get a user's Policy, using a currently connected ServiceBase. Use class constants, defined in the PolicyKeys interface to pull specific policy values. 
        RoomServices roomSvcs = new RoomServices(SERVER_URL);
        Policy policy = Policy.getPolicyService(roomSvcs);
        String stringValue = policy.getStringValue(Policy.);
        boolean booleanValue = policy.getBooleanValue(Policy.);
        int intValue = policy.getIntValue(Policy.);

Parameters:

String - id - This is the userId, or companyId, based on the Policy.Type parameter.
type - the type of id being passed in.
Returns:

PolicyException - If the server was reached, but some policy specific problem occurred.
