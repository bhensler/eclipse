com.lotus.sametime.chatui

Interface ChatUI

  • All Superinterfaces:
    STCompApi
    All Known Implementing Classes:
    ChatUIComp
    Deprecated. 
    due to the availability of additional non-AWT GUI frameworks. This component is no longer supported.

    public interface ChatUI
extends STCompApi
    The interface defines the Instant Message (IM) manager UI component. This component manages the creation of 1-on-1 chats (IM), chat confrences and meetings.
