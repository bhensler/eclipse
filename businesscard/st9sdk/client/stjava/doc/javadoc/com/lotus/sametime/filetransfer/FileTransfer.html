      • setUpdateInterval

        public void setUpdateInterval(int interval)
        Sets the update interval for sending notifications about the number of bytes transferred in the current file transfer. The interval is specified as a percentage of the total number of bytes. A notification will be sent to the listener each time a number of bytes equal to the specified percentage has been sent.
        Parameters:
        interval - Percentage value. Must be in the range of 1 to 100. Values outside the range are ignored, leaving the previous interval unchanged.
