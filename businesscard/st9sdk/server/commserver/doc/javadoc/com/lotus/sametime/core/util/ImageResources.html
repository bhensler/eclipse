      • getImage

        public java.awt.Image getImage(java.lang.String imageName)
        Get an image resource specified by its file name. In application mode the file should be located in the same path as the class files(e.g. jar file, system CLASSPATH etc). In an applet mode loaded on a Netsacpe navigator the images have to be compressed into a jar file, and the jar file has to be specified in the html applet tag.
        Parameters:
        imageName - The absolute file name (/file.gif).
        Returns:
        The requested image as an java.awt.Image object, or null if the image can't be loaded for some reason.
